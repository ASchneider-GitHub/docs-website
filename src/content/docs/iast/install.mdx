<Callout variant="important">
    Run IAST with non-production deployments only to avoid exposing vulnerabilities on your production software.

    IAST tests your applications for any exploitable vulnerability. It replays the generated HTTP request with vulnerable payloads.
</Callout>

To install New Relic IAST:

1. Go to **[one.newrelic.com](https://one.newrelic.com) > All capabilities > IAST** and click **Set up IAST with applications**.

    <img
        title="Set up IAST with applications"
        alt="Set up IAST with applications"
        src={iastSetup}
    />

2. In the installation window, select the language of your application and complete the steps.

    <img
        title="Install New Relic IAST"
        alt="Install New Relic IAST"
        src={iastInstall}
    />

3. Make sure your `newrelic.yml` config file is updated as follows:

    ```yml
    host: staging-collector.newrelic.com
    security:
        enabled: true
    agent:
        enabled: true
    validator_service_url: wss://csec-staging.nr-data.net
    ```

4. Once you've completed all the steps, restart your application and generate traffic against the application's APIs.

5. Click **See your data** to see an overview of your tested application.
