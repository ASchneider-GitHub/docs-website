## StepsItem

Child element of the `<Steps>` component.

### Usage

```
import { StepsItem } from 'nr1'
```



### Props

|                                    |                                                                                                                                                                                                                                                                                                                                                                    |
| ---------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `ariaLabel`##### string            | Provide a label for screen readers to describe the step. This is only required if the screen reader label should be different to the `label` prop.                                                                                                                                                                                                                 |
| `checked`##### boolean             | If `true`, marks the step as completed.                                                                                                                                                                                                                                                                                                                            |
| `children`##### REQUIRED##### node | Content to render inside the item.                                                                                                                                                                                                                                                                                                                                 |
| `className`##### string            | Appends class names to the component.Should be used only for positioning and spacing purposes.                                                                                                                                                                                                                                                                     |
| `disabled`##### boolean            | Use disabled to prevent users from opening the step by clicking on its label.                                                                                                                                                                                                                                                                                      |
| `expanded`##### boolean            | When true, it will show the content of the step.                                                                                                                                                                                                                                                                                                                   |
| `label`##### REQUIRED##### string  | Label of the step.                                                                                                                                                                                                                                                                                                                                                 |
| `onClick`##### function            |                                                                                                                                                                                                                                                                                               |
| `style`##### object                | Inline style for custom styling.Should be used only for positioning and spacing purposes.                                                                                                                                                                                                                                                                          |
| `testId`##### string               | Adds a `data-test-id` attribute. Use it to target the component in unit and E2E tests.For a test id to be valid, prefix it with your nerdpack id, followed up by a dot.For example, `my-nerdpack.some-element`.**Note:** You might not see `data-test-id` attributes as they are removed from the DOM, to debug them pass a `e2e-test` query parameter to the URL. |
| `value`##### any                   | Used by the `<Steps>` to identify the active step.                                                                                                                                                                                                                                                                                                                 |
