## Icon

React component used for rendering icons consistently through the UI. The icon component supports various icons in different sizes, and it also allows you to specify an icon color.

### Usage

```
import { Icon } from 'nr1'
```



### Examples

#### Basic

```
<Icon type={Icon.TYPE.HARDWARE_AND_SOFTWARE__SOFTWARE__PLUGIN__S_OK} />
```
