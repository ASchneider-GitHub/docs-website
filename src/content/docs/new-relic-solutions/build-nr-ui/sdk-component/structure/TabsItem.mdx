## TabsItem

Child element of the `<Tabs>` component.

It creates a tab where the title of the tab is defined by the `label` prop and the content of the tab is defined by the `children` prop.

### Usage

```
import { TabsItem } from 'nr1'
```



### Props

|                                    |                                                                                                                                                                                                                                                                                                                                                                    |
| ---------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `children` <h5>REQUIRED</h5> <h5>node</h5> | Content to show when the tab is selected.                                                                                                                                                                                                                                                                                                                          |
| `className` <h5>string</h5>            | Appends class names to the component.Should be used only for positioning and spacing purposes.                                                                                                                                                                                                                                                                     |
| `disabled` <h5>boolean</h5>            | If `true`, the tab is not available for interaction.                                                                                                                                                                                                                                                                                                               |
| `label` <h5>REQUIRED</h5><h5>string</h5>  | Title of the tab.                                                                                                                                                                                                                                                                                                                                                  |
| `style` <h5>object</h5>                | Inline style for custom styling.It is applied to the tab content.                                                                                                                                                                                                                                                                                                  |
| `testId` <h5>string</h5>               | Adds a `data-test-id` attribute. Use it to target the component in unit and E2E tests.For a test id to be valid, prefix it with your nerdpack id, followed up by a dot.For example, `my-nerdpack.some-element`.**Note:** You might not see `data-test-id` attributes as they are removed from the DOM, to debug them pass a `e2e-test` query parameter to the URL. |
| `value` <h5>REQUIRED</h5> <h5>any</h5>     | Unique identifier of the tab.It is used when selecting the active tab.                                                                                                                                                                                                                                                                                             |
