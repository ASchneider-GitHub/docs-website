A new version of the <DNT>Fluent Bit</DNT> package has been released with a fix for [CVE-2024-4323](https://nvd.nist.gov/vuln/detail/CVE-2024-4323). Follow the recommended procedures to [update the Infrastructure agent](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent/): 

<table>

    <tbody>
        <tr>
            <td>
                Windows Infrastructure agent
            </td>
            <td>
                On Windows, the embedded version of <DNT>Fluent Bit</DNT> within the Windows Infrastructure Agent has been determined to not be impacted by CVE-2024-4323.
            </td>
        </tr>
        <tr>
            <td>
                Linux Infrastructure agent
            </td>
            <td>
                Upgrade the Infrastructure agent to version 1.52.3 or later, AND update <DNT>Fluent Bit</DNT> to version 3.0.4 or later 
            </td>
        </tr>
    </tbody>
</table>

## Changed

* Update <DNT>Fluent Bit</DNT> to version 3.0.4 as a fix for [CVE-2024-4323](https://nvd.nist.gov/vuln/detail/CVE-2024-4323), which impacted certain prior <DNT>Fluent Bit</DNT> versions contained in the Infra-agent log forwarder solution.
* Bump nri-flex to v1.13.0 (https://github.com/newrelic/infrastructure-agent/pull/1860)
