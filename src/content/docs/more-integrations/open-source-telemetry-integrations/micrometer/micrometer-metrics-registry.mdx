You can configure the OpenTelemetry (OTel) bridge to forward Micrometer data to New Relic. The OTel Micrometer bridge directs Micrometer to record telemetry to OTel, which you can then export to New Relic with the OpenTelemtry SDK. 

This doc walks you through procedures to forward Micrometer data with an example Spring app. These are generalized steps that you may want to update to fit your specific environment. 

## Prerequisites [#prereqs]

Before you get started, we recommend that you:

* A Springboot app with Micrometer enabled. Your Springboot app should be instrumented with Micrometer out of the box.
* Review the [Getting started](https://docs.newrelic.com/docs/style-guide/capitalization/product-capability-feature-usage/) section of the OpenTelemetry Java instrumentation repo. 
* At minimum, ensure your OTel Java agent is the latest version.

## Forward Micrometer data to New Relic [#forward-data-otel]

<Steps>
    <Step>
### Add OpenTelemetry Micrometer instrumentation

Add OTel Micrometer instrumentation to the alpha modules section of your `build.gradle` file:

    ```gradle
    implementation 'io.opentelemetry.instrumentation:opentelemetry-micrometer-1.5'
    ```

    </Step>
    <Step>

        ### Add OTel API and SDK dependencies
        
In the dependencies argument, add the OTel API and SDK:

    ```
    implementation 'io.opentelemetry:opentelemetry-api'
    implementation 'io.opentelemetry:opentelemetry-sdk'
    ```
    
An example completed file might look something like this:

    ```
    import org.springframework.boot.gradle.plugin.SpringBootPlugin

    plugins {
    id 'java-library'
    id 'org.springframework.boot'
    }

    bootRun {
    mainClass.set 'io.opentelemetry.example.micrometer.Application'
    }

    dependencies {
    //api and sdk dependencies
    implementation 'io.opentelemetry:opentelemetry-api'
    implementation 'io.opentelemetry:opentelemetry-sdk'

    //alpha modules with micrometer instrumentation
    implementation 'io.opentelemetry.instrumentation:opentelemetry-micrometer-1.5'
    implementation 'io.opentelemetry:opentelemetry-exporter-otlp'

    //spring modules
    implementation platform(SpringBootPlugin.BOM_COORDINATES)
    implementation 'org.springframework.boot:spring-boot-starter-web'
    implementation 'org.springframework.boot:spring-boot-starter-actuator'
    implementation 'org.springframework.boot:spring-boot-starter-aop'
    }
    ```
