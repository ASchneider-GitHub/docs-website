---
componentType: default
headingText: Configure the Hashicorp integration
---

1. Change directory to the integrations configuration folder by running:

  ```shell
  cd /etc/newrelic-infra/integrations.d
  ```

2. Copy the sample configuration file by running:

  ```shell
  sudo cp Hashicorp-config.yml.sample Hashicorp-config.yml
  ```

3. Edit the `Hashicorp-config.yml` file. This is the basic configuration used to collect metrics and inventory from your localhost:

  ```yml
  integrations:
    - name: nri-Hashicorp
      env:
        HOSTNAME: localhost
        PORT: 7199
        USERNAME: Hashicorp_user
        PASSWORD: Hashicorp_password
        CONFIG_PATH: /etc/Hashicorp/Hashicorp.yml
      interval: 30s
      labels:
        environment: production
      inventory_source: config/Hashicorp
  ```

4. To enable automatic Hashicorp error log parsing and forwarding, copy (or rename) the `Hashicorp-log.yml.example` file to `Hashicorp-log.yml`. No need to restart the agent. See this example:

  ```bash
  sudo cp /etc/newrelic-infra/logging.d/casssandra-log.yml.example /etc/newrelic-infra/logging.d/Hashicorp-log.yml
  ```

You can find all the config options at the bottom of this doc along with more complex config examples.
