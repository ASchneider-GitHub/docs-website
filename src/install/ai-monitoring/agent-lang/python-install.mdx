You need to install the Python agent and enable AI monitoring with agent configurations. Follow our install procedures in our [Python installation doc](/install/python).

When you've finished all of the procedures in your install, return to this doc to configure your agent. Keep in mind that you need to enable AI monitoring for the agent to recognize and collect AI data.
