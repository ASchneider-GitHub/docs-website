Before installation, ensure you meet these requirements:

* A New Relic account. Don't have one? [Sign up for free!](https://newrelic.com/signup) No credit card required.
* Apache Flink versions 2.2 and 2.4
* [Apache Flink status module](https://httpd.Apache Flink.org/docs/2.4/mod/mod_status.html) enabled and configured for Apache Flink instance.
* Apache Flink status module endpoint (default `server-status`) available from the host containing the Apache Flink integration.
