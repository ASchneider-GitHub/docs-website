componentType: default
headingText: Configure the Memcached integration
---

1. In the Integrations directory, `C:\Program Files\New Relic\newrelic-infra\integrations.d\`, create a copy of the sample configuration file by running:

```shell
copy Memcached3-config.yml.sample Memcached3-config.yml
```

2. Edit the `Memcached3-config.yml` file. The only two required fields are `Memcached_cluster_name` and `Memcached_uri`. This example config file collects all metrics:

```yml
integrations:
- name: nri-memcached3
    config:
    Memcached_cluster_name: my_cluster
    Memcached_uri: Memcached://username:password@localhost:27017
    exporter_port: 9126
```

You can find all the config options at the bottom of this doc along with more complex config examples.
