Follow these instructions to add pre-defined dashboards:

1. Go to <DoNotTranslate>**[one.newrelic.com](https://one.newrelic.com) > +Add data**</DoNotTranslate> and type <DoNotTranslate>**Kubernetes**</DoNotTranslate>.

2. Under <DoNotTranslate>**Dashboards**</DoNotTranslate>, click <DoNotTranslate>**Kubernetes**</DoNotTranslate>.

    <img
    title="Add Kubernetes dashboards"
    alt="Add Kubernetes dashboards"
    src={kubernetesAddDashboards}
    />

3. Click <DoNotTranslate>**Edit**</DoNotTranslate> if you want to change the account in the popup window that opens.

4. Click <DoNotTranslate>**Setup Kubernetes**</DoNotTranslate> or <DoNotTranslate>**Skip this step**</DoNotTranslate> if you already setup this datasource. 

5. Click <DoNotTranslate>**View dashboard**</DoNotTranslate>, and see your Kubernetes data in New Relic.


You can also add pre-defined alert policies:

1. Go to <DoNotTranslate>**[one.newrelic.com](https://one.newrelic.com) > +Add data**</DoNotTranslate> and type <DoNotTranslate>**Kubernetes**</DoNotTranslate>.

2. Under <DoNotTranslate>**Alerts**</DoNotTranslate> select one of the options.

    <img
    title="Add Kubernetes alerts"
    alt="Add Kubernetes alerts"
    src={kubernetesAddAlerts}
    />

3. Click <DoNotTranslate>**Edit**</DoNotTranslate> if you want to change the account in the popup window that opens.

4. Click <DoNotTranslate>**Setup Kubernetes**</DoNotTranslate> or <DoNotTranslate>**Skip this step**</DoNotTranslate> if you already setup this datasource. 

5. Click <DoNotTranslate>**View alert policy**</DoNotTranslate>, and see your Kubernetes data in New Relic.
