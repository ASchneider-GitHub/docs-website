headingText: Find and use data
---

1. Go to <DNT>**[one.newrelic.com](https://one.newrelic.com) > Integrations & Agents**</DNT>. 

2. Select <DNT>**Dashboards**</DNT> to access the pre-built resources.

3. Search <DNT>**MongoDB**</DNT> and select the dashboard.

  <img
    title="Install the MongoDB dashboard" 
    alt="Installing the MongoDB dashboard" 
    src="/images/infrastructure_screenshot-crop_mongodb-dashboard..webp"
  />

4. Click <DNT>**Edit**</DNT> if you want to change the account in the popup window that opens.

5. Click <DNT>**Setup MongoDB**</DNT> or <DNT>**Skip this step**</DNT> if you already setup this integration.

For more on how to find and use your data, see how to [understand integration data](/docs/infrastructure/integrations/find-use-infrastructure-integration-data). You can find a full list of metrics and their attributes at the bottom of this doc.
