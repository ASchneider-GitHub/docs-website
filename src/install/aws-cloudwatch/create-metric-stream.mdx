To stream CloudWatch metrics to New Relic:

Check the minimal permissions and mapping instructions.
Create Kinesis Data Firehose and point it to New Relic.
Next, create a CloudWatch Metric Stream to send metrics to that Firehose you've just created.
Follow the guided or manual setup instructions.
Validate data reception.
If applicable, read our documentation about migrating from AWS polling integrations.
