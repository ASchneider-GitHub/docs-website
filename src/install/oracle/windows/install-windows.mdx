componentType: default
headingText: Configure the MongoDB integration
---

1. In the Integrations directory, `C:\Program Files\New Relic\newrelic-infra\integrations.d\`, create a copy of the sample configuration file by running:

```
copy oracledb-config.yml.sample oracledb-config.yml
```

2. Edit the `oracledb-config.yml` file. The following configuration collects metrics and inventory from your localhost. It uses the default connection on port 1521. Do not forget to replace `SERVICE_NAME` and `ORACLE_HOME` with the correct values for your environment:

```
integrations:
  - name: nri-oracledb
    env:
      SERVICE_NAME: ORACLE
      HOSTNAME: 127.0.0.1
      PORT: 1521
      USERNAME: oracledb_user
      PASSWORD: oracledb_password
      ORACLE_HOME: /app/oracle/product/version/database
    interval: 15s
    labels:
      environment: production
    inventory_source: config/oracledb
```
You can find all the config options at the bottom of this doc along with more complex config examples.
