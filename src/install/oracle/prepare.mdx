headingText: Prepare your Oracle database
---

Follow these steps to get your Oracle data base set up for our integration:

1. Configure `ORACLE_HOME` on your Oracle database for the root user. See [Oracle's docs for more info](https://docs.oracle.com/database/121/ADMQS/GUID-EC18C4A6-3BA5-4C14-9D76-B0DD62FEFFF2).
2. Configure `listener.ora` on your Oracle database to monitor remote connections. By default, Oracle Database only listens to localhost. See [https://docs.oracle.com/en/database/oracle/oracle-database/23/netag/configuring-and-administering-oracle-net-listener.html#GUID-C3C40DBA-4282-41E1-9562-4B8B10947C4E](Oracle's docs for more info).
