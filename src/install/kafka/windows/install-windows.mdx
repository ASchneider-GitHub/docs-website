componentType: default
headingText:  Install the Kafka monitoring integration
---

1. Download the [.exe installer for New Relic's Kafka integration](https://download.newrelic.com/infrastructure_agent/windows/integrations/nri-kafka/nri-kafka-amd64-installer.exe).

2. In your terminal, run this command:

  ```shell
    PATH\TO\nri-kafka-amd64-installer.exe
  ```

3. Go to the integrations directory, `C:\Program Files\New Relic\newrelic-infra\integrations.d\`. Run this command to create a copy of the sample configuration file:

  ```shell
    cp kafka-config.yml.sample kafka-config.yml
  ```

4. Edit the `kafka-config.yml` file by using one of the [`kafka-config.yml` sample files](#examples).
