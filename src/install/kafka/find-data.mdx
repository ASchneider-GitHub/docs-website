headingText: Find and use data
---

1. Go to <DNT>**[one.newrelic.com](https://one.newrelic.com) > Integrations & Agents**</DNT> .

2. Select <DNT>**Dashboards**</DNT> to access the pre-built resources. 

3. Search <DNT>**Kafka**</DNT> and select the dashboard.

    <img
      title="Add Kafka dashboard"
      alt="Add Kafka dashboard"
      src="/images/infrastructure_screenshot-crop_kafka-adding-dashboard.webp"
    />

4. Click <DNT>**Edit**</DNT> if you want to change the account in the popup window that opens.

5. Click <DNT>**Setup Kafka**</DNT> or <DNT>**Skip this step**</DNT> if you already setup this integration.

6. Click <DNT>**View dashboard**</DNT>, and see your Kafka data in New Relic.


Kafka data is attached to the following [event types](/docs/data-apis/understand-data/new-relic-data-types/#events-new-relic):

* [`KafkaBrokerSample`](#broker-sample)
* [`KafkaTopicSample`](#topic-sample)
* [`KafkaProducerSample`](#producer-sample)
* [`KafkaConsumerSample`](#consumer-sample)
* [`KafkaOffsetSample`](#offset-sample)

You can [query this data](/docs/query-your-data/explore-query-data/get-started/introduction-querying-new-relic-data) for troubleshooting purposes or to create [charts](/docs/query-your-data/explore-query-data/use-charts/use-your-charts) and [dashboards](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards).

For more on data reported from infrastructure integrations, see how to [Understand and use data from infrastructure integrations](/docs/infrastructure/infrastructure-data/infra-integration-data).
