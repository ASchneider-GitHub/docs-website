1. Go to **[one.newrelic.com](https://one.newrelic.com) > Add data** and type **Elasticsearch**.
2. Under **Dashboards**, click **Elasticsearch overview dashboard**.
3. In the popup window, select your account
4. Click **View dashboard**, and see your Elasticsearch data in New Relic.

For more on how to find and use your data, see how to [understand integration data](/docs/infrastructure/integrations/find-use-infrastructure-integration-data). You can find a full list of metrics and their attributes at the bottom of this doc.
