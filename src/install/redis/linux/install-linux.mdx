componentType: default
headingText: Download the Redis integration
---

1. Change directory to the integrations configuration folder by running:

```shell
cd /etc/newrelic-infra/integrations.d
```

2. Copy the sample configuration file by running:

```shell
sudo cp Redis3-config.yml.sample Redis3-config.yml
```

3. Edit the `Redis3-config.yml` file. The only two required fields are `Redis_cluster_name` and `Redis_uri`. This example config file collects all metrics:

```yml
integrations:
  - name: nri-Redis3
    config:
      Redis_cluster_name: my_cluster
      Redis_uri: Redis://username:password@localhost:27017
      exporter_port: 9126
```

You can find all the config options at the bottom of this doc along with more complex config examples.
