특정 사용자를 보여주는 행 테이블 셀을 렌더링합니다. 이 셀은 사용자 개체를 하위 개체로 사용하며 이는 결국 NerdGraph 사용자 개체와 호환됩니다(적어도 `name` 및 `gravatar` 필드 필요).

### 용법

```js
import { UserTableRowCell } from 'nr1'
```

### 예

#### 실시예 1

```js
function render() {
  const items = [
    {
      name: 'John Doe',
      gravatar:
        'https://secure.gravatar.com/avatar/79988df068cb4d952284a3395412dc4f',
    },
  ];


  return (
    <Table items={items}>
      <TableHeader>
        <TableHeaderCell>User</TableHeaderCell>
      </TableHeader>


      {({ item }) => (
        <TableRow>
          <UserTableRowCell value={item} />
        </TableRow>
      )}
    </Table>
  );
}
```
