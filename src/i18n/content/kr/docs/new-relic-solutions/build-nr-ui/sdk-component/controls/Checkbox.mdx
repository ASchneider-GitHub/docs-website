사용자가 제공된 하나 이상의 옵션을 선택할 때

체크박스 상태는 `checked` 및 `indeterminate` 속성을 통해 제어할 수 있습니다. 둘 중 어느 것도 제공되지 않으면 확인란이 자체 상태를 관리합니다.

`onChange` 소품을 통해 체크박스 상태 변경을 들을 수 있습니다.

**참고**: `checked` 설정은 구성요소를 제어된 상태로 전환하므로 `defaultChecked` 재정의합니다. `defaultChecked` 제어되지 않는 확인란에만 사용해야 합니다.

### 용법

```js
import { Checkbox } from 'nr1'
```

### 예

#### 기초적인

```js
<div className="nr1-Docs-prettify">
  <Checkbox onChange={(event) => alert('Foo')} label="Foo" />
  <Checkbox indeterminate label="Bar" />
  <Checkbox checked disabled label="Baz" />
</div>
```

#### 정보 포함

```js
<Checkbox onChange={(event) => alert('Foo')} info="Info value" label="Foo" />
```

#### 설명 포함

```js
<Checkbox
  onChange={(event) => alert('Foo')}
  description="Description value"
  label="Foo"
/>
```

#### 잘못된 메시지 포함

```js
<Checkbox
  onChange={(event) => alert('Foo')}
  invalid="Invalid message value"
  label="Foo"
/>
```

#### 제어되는 구성 요소

```js
class MyNerdlet extends React.PureComponent {
  constructor(props) {
    super(props);


    this.state = {
      isChecked: false,
    };


    this.onChange = this.onChange.bind(this);
  }


  onChange(event) {
    console.log(event);
    const isChecked = event.target.checked;


    this.setState({ isChecked });
  }


  render() {
    return (
      <Checkbox
        checked={this.state.isChecked}
        onChange={this.onChange}
        label="Foo"
      />
    );
  }
}
```
