`<AutoSizer>` 구성요소는 포함된 요소의 크기 변화를 측정하여 이에 반응하는 기능을 제공합니다. 해당 외부 요소의 `width` 또는 `height` 가 변경되면 하위 요소가 다시 렌더링되고 새 측정값을 객체로 수신합니다.

### 용법

```js
import { AutoSizer } from 'nr1'
```

### 예

#### 기초적인

```js
function render() {
  const style = {
    backgroundColor: '#d8211a1a',
    border: 'dashed 1px #d8211a4d',
  };


  return (
    <AutoSizer>
      {({ width, height }) => (
        <div style={{ ...style, height, width }}>
          width: {width}, height: {height}
        </div>
      )}
    </AutoSizer>
  );
}
```
