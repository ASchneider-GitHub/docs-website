단일 행을 구성하는 셀의 래퍼입니다.

### 용법

```js
import { TableRow } from 'nr1'
```

### 소품

<table>
  <tbody>
    <tr>
      <td>
        `actions` <h5>모양\[]</h5>
      </td>

      <td>
        사용자가 행을 가리키면 표시될 작업 집합을 정의합니다. 작업에는 필수 텍스트와 onClick 콜백(해당 항목이 속한 행의 항목 및 인덱스와 함께 호출됨)이 있으며 비활성화된 속성도 허용합니다.

        <h3>모양</h3>

        `disabled` <h5>부울</h5>

        `label` <h5>필수</h5><h5>문자열 행</h5> 의 작업을 설명하는 데 사용합니다.

        `onClick` <h5>필수</h5><h5>함수</h5> 콜백은 사용자가 작업을 클릭할 때마다 실행됩니다.

        <FunctionDefinition
          returnValue={[]}
          arguments={[{"name":"event","type":"React.MouseEvent","description":""}]}
        />

        `testId` <h5>문자열</h5> `data-test-id` 속성을 추가합니다. 이를 사용하여 단위 및 E2E 테스트의 구성 요소를 목표로 삼습니다. 테스트 ID가 유효하려면 앞에 nerdpack ID를 붙이고 그 뒤에 점이 옵니다. 예를 들어 `my-nerdpack.some-element`. **참고:** `data-test-id` 속성이 DOM 에서 제거될 때 표시되지 않을 수 있습니다. 이를 디버그하려면 `e2e-test` 쿼리 변수를 URL에 전달해야 합니다.

        `type` <h5>enum</h5> 액션 드롭다운 메뉴의 각 항목에 대한 액션 유형입니다. 필수는 아니며 기본값은 `TableRow.ACTION_TYPE.NORMAL` 입니다.

        * 일반 — 일반적이고 중요하지 않은 작업입니다.

        * 파괴적 — 사용자가 완료하기 전에 일시 중지하고 고려하도록 하는 삭제 또는 제거와 같은 파괴적인 작업이 있을 때 사용합니다.

          <OptionReference>
            TableRow.ACTION_TYPE.DESTRUCTIVE, TableRow.ACTION_TYPE.NORMAL,
          </OptionReference>

        ```js
        const actions = [
          {
            label: 'Duplicate',
            type: TableRow.ACTION_TYPE.NORMAL,
          },
          {
            label: 'Delete',
            type: TableRow.ACTION_TYPE.DESTRUCTIVE,
          },
        ];
        ```

        ```js
        const actions = [
          {
            label: 'Action 1',
            disabled: true,
            onClick: (evt, { item, index }) => {
              alert(`Action 1:Item: ${index}${JSON.stringify(item, null, 2)}`);
            },
          },
          {
            label: 'Action 2',
            onClick: (evt, { item, index }) => {
              alert(`Action 2:Item: ${index}${JSON.stringify(item, null, 2)}`);
            },
          },
        ];
        ```
      </td>
