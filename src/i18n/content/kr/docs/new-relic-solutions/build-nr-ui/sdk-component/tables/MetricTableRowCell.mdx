특정 지표를 표시하는 행 테이블 셀을 렌더링합니다. 지표는 전달된 단위에 따라 형식이 지정됩니다. 단위는 `<NrqlQuery>` 형식 유형 `CHART` 에서 지원하는 단위와 1:1 호환됩니다.

단위 및 접두사 형식은 플랫폼의 나머지 부분과 일치하도록 자동으로 선택됩니다. 지표는 자동으로 왼쪽으로 정렬되고 오른쪽에서 생략됩니다. 이는 지표를 표시하는 표준 방법입니다.

### 용법

```js
import { MetricTableRowCell } from 'nr1'
```

### 예

#### 기초적인

```js
function render() {
  const items = [
    {
      value: 123456789,
    },
  ];


  return (
    <Table items={items}>
      <TableHeader>
        <TableHeaderCell alignmentType={TableHeaderCell.ALIGNMENT_TYPE.RIGHT}>
          Memory usage
        </TableHeaderCell>
      </TableHeader>


      {({ item }) => (
        <TableRow>
          <MetricTableRowCell
            type={MetricTableRowCell.TYPE.BITS}
            value={item.value}
          />
        </TableRow>
      )}
    </Table>
  );
}
```

#### 추가적인 가치로

```js
function render() {
  const items = [
    {
      value: 123456789,
      lastUpdated: new Date().toLocaleTimeString(),
    },
  ];


  return (
    <Table items={items} multivalue>
      <TableHeader>
        <TableHeaderCell alignmentType={TableHeaderCell.ALIGNMENT_TYPE.RIGHT}>
          Memory usage
        </TableHeaderCell>
      </TableHeader>


      {({ item }) => (
        <TableRow>
          <MetricTableRowCell
            type={MetricTableRowCell.TYPE.BITS}
            value={item.value}
            additionalValue={`Last updated at ${item.lastUpdated}`}
          />
        </TableRow>
      )}
    </Table>
  );
}
```
