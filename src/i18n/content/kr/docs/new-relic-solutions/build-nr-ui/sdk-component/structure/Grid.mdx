그리드는 2차원 레이아웃 시스템입니다. 즉, 열과 행을 모두 처리할 수 있어 주로 1차원 시스템인 `<Stack>` 과 달리 그리드 기반 사용자 인터페이스를 디자인하는 데 도움이 됩니다.

### 용법

```js
import { Grid } from 'nr1'
```
