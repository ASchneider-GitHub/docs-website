`<CardBody>` 구성요소의 하위 요소입니다.

### 용법

```js
import { CardSection } from 'nr1'
```

### 소품

<table>
  <tbody>
    <tr>
      <td>
        `children` <h5>필수</h5> <h5>노드</h5>
      </td>

      <td>
        카드 내부에 렌더링할 콘텐츠입니다.
      </td>
    </tr>

    <tr>
      <td>
        `className` <h5>끈</h5>
      </td>

      <td>
        구성 요소에 클래스 이름을 추가합니다. 위치 지정 및 간격 지정 목적으로만 사용해야 합니다.
      </td>
    </tr>

    <tr>
      <td>
        `collapsed` <h5>부울</h5>
      </td>

      <td>
        이 소품을 설정하여 구성 요소의 접힌 상태를 제어합니다. 정의된 경우 구성 요소를 [제어되는 구성 요소](https://facebook.github.io/react/docs/forms.html) 로 바꿉니다.
      </td>
    </tr>

    <tr>
      <td>
        `collapsible` <h5>부울</h5>
      </td>

      <td>
        구성 요소를 접을 수 있도록 하려면 이 소품을 설정합니다.
      </td>
    </tr>

    <tr>
      <td>
        `defaultCollapsed` <h5>부울</h5>
      </td>

      <td>
        `true` 인 경우 카드의 초기 접힌 상태는 `collapsed` 입니다. [제어되는 구성요소를](https://facebook.github.io/react/docs/forms.html) 사용하지 않을 때 유용합니다.
      </td>
    </tr>

    <tr>
      <td>
        `onChange` <h5>기능</h5>
      </td>

      <td>
        이 함수는 사용자가 구성 요소를 축소하거나 확장하기 위해 헤더를 클릭할 때마다 호출됩니다. 구성요소의 상태를 제어하는 경우 이를 사용하여 `collapsed` 소품의 값을 설정합니다.

        <FunctionDefinition
          returnValue={[]}
          arguments={[{"name":"event","type":"React.MouseEvent","description":""},{"name":"collapsed","type":"boolean","description":""}]}
        />
      </td>
    </tr>

    <tr>
      <td>
        `style` <h5>물체</h5>
      </td>

      <td>
        사용자 정의 스타일을 위한 인라인 스타일입니다. 위치 지정 및 간격 지정 목적으로만 사용해야 합니다.
      </td>
    </tr>

    <tr>
      <td>
        `testId` <h5>끈</h5>
      </td>

      <td>
        `data-test-id` 속성을 추가합니다. 이를 사용하여 단위 및 E2E 테스트의 구성 요소를 목표로 삼습니다. 테스트 ID가 유효하려면 앞에 nerdpack ID를 붙이고 그 뒤에 점이 옵니다. 예를 들어 `my-nerdpack.some-element`.

        **참고:** `data-test-id` 속성이 DOM 에서 제거될 때 표시되지 않을 수 있습니다. 이를 디버그하려면 `e2e-test` 쿼리 변수를 URL에 전달해야 합니다.
      </td>
    </tr>
  </tbody>
</table>
