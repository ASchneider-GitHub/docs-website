테이블 본문에 있는 테이블 셀의 내용을 래핑합니다. 이 구성요소는 `<TableRow>` 의 하위 요소로만 사용될 것으로 예상됩니다. 셀은 맞지 않는 내용을 타원으로 표시하고 수직으로 중앙에 배치하는 작업을 자동으로 처리합니다.

이 구성 요소는 기본 셀입니다. 일반적인 경우 SDK는 UI 전체에서 항상 일관되게 그려지도록 추가 셀 세트도 제공합니다(예: 상태가 있는 엔터티 이름 또는 사용자).

### 용법

```js
import { TableRowCell } from 'nr1'
```

### 예

#### 외부 링크와 추가 정보가 있는 행 셀

```js
function render() {
  const items = [
    {
      dashboardName: 'Staging service - Errors',
      createdBy: 'john@doe.com',
      url: '//fake-url.com/dashboard1',
    },
    {
      dashboardName: 'Package Registry - Performance',
      createdBy: 'jane@doe.com',
      url: '//fake-url.com/dashboard2',
    },
  ];
  return (
    <Table items={items} multivalue>
      <TableHeader>
        <TableHeaderCell>Name</TableHeaderCell>
        <TableHeaderCell>Url</TableHeaderCell>
      </TableHeader>
      {({ item }) => (
        <TableRow>
          <TableRowCell additionalValue={`Created by ${item.createdBy}`}>
            {item.dashboardName}
          </TableRowCell>
          <TableRowCell>
            <Link to={item.url}>Go to the dashboard</Link>
          </TableRowCell>
        </TableRow>
      )}
    </Table>
  );
}
```
