특정 엔터티의 제목을 표시하는 행 테이블 셀을 렌더링합니다. 이 셀은 엔터티 개체를 하위 요소로 사용하며 이는 결국 NerdGraph 엔터티 개체와 호환됩니다(적어도 `name` 및 `reporting` 필드 필요). 경고 가능한 경우 `alertSeverity` 도 쿼리해야 합니다.

### 용법

```js
import { EntityTitleTableRowCell } from 'nr1'
```

### 예

#### 기초적인

```js
function render() {
  const items = [
    {
      name: 'Login service',
      guid: 'MTIzNDU2fEZPT3xCQVJ8OTg3NjU0MzIx',
      alertSeverity: 'CRITICAL',
      reporting: true,
    },
  ];


  return (
    <Table items={items}>
      <TableHeader>
        <TableHeaderCell>Entity</TableHeaderCell>
      </TableHeader>


      {({ item }) => (
        <TableRow>
          <EntityTitleTableRowCell value={item} />
        </TableRow>
      )}
    </Table>
  );
}
```

#### 추가적인 가치로

```js
function render() {
  const items = [
    {
      name: 'Login service',
      guid: 'MTIzNDU2fEZPT3xCQVJ8OTg3NjU0MzIx',
      alertSeverity: 'CRITICAL',
      reporting: true,
    },
  ];


  return (
    <Table items={items} multivalue>
      <TableHeader>
        <TableHeaderCell>Entity</TableHeaderCell>
      </TableHeader>


      {({ item }) => (
        <TableRow>
          <EntityTitleTableRowCell value={item} additionalValue={item.guid} />
        </TableRow>
      )}
    </Table>
  );
}
```
