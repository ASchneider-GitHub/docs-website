래핑된 구성요소에 간격을 제공합니다. 간격은 0, 1, 2, 3 또는 4개의 값을 포함할 수 있는 튜플로 제공됩니다. 이러한 각 값은 TYPE 정적 속성에서 나옵니다.

튜플의 구조는 `margin` 또는 `padding` 속성이 작동하는 방식과 매우 유사합니다. 하나의 값을 지정하면 값이 4개 면에 복제되고 두 개는 `top`/`bottom` 및 `left`/`right` 등에 복제됩니다. 특정 값을 생략하고 다른 값은 전달하려는 경우 `OMIT` 플레이스홀더로 사용할 수 있습니다. 사용되는 측면(또는 측면)의 공간은 그대로 유지됩니다.

SDK의 다른 구성 요소에는 간격 기능이 내장되어 있습니다. 자체 구성요소에 동일한 기능을 복제하려면 `Spacing.add` 사용하고 구성요소를 전달하면 됩니다. 메서드에서 반환된 구성 요소에는 정적 요소뿐만 아니라 `type` 소품도 포함됩니다.

### 용법

```js
import { Spacing } from 'nr1'
```
