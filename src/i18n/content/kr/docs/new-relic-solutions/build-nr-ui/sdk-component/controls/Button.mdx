버튼은 사용자에게 작업을 완료하는 방법을 제공하기 위해 제품 전반에 걸쳐 사용됩니다. 경험상 버튼을 과도하게 사용하지 마십시오. 이렇게 하면 사용자가 어떤 조치를 취하기를 원하는지 정확히 알 수 있습니다. 경험상 기본 버튼을 두 개 이상 사용하지 마십시오.

주요 기능 — 버튼은 다음을 수행할 수 있습니다.

* 하나의 작업을 수행
* 여러 개의 버튼이 있는 경우 여러 작업을 수행합니다.

### 용법

```js
import { Button } from 'nr1'
```

### 예

#### 기초적인

```js
<div className="nr1-Docs-prettify">
  <Button onClick={() => alert('Hello World!')} type={Button.TYPE.PRIMARY}>
    Click me
  </Button>
  <Button
    onClick={() => alert('Hello World!')}
    type={Button.TYPE.PRIMARY}
    iconType={
Button.ICON_TYPE.DOCUMENTS__DOCUMENTS__NOTES__A_ADD}
  >
    Click me
  </Button>
  <Button
    type={Button.TYPE.PRIMARY}
    iconType={
Button.ICON_TYPE.DOCUMENTS__DOCUMENTS__FILE__A_ADD}
  />
</div>
```

      <td>
        패널을 확장하거나 여는 데 사용될 때 `Button` 가 제어하는 요소의 `id` 문자열을 전달합니다. `ariaExpanded` 와 함께 사용하세요.

        ```js
        const { expanded } = this.state;
        const panelId = 'content-panel';

        return (
          <>
            <Button ariaControls={panelId} ariaExpanded={expanded}>
              Expand content
            </Button>
            <div id={panelId} hidden={!expanded}>
              Content
            </div>
          </>
        );
        ```
      </td>
    </tr>

      </td>

      <td>
        이를 사용하여 `Button` 제어 요소가 확장되었는지 여부를 나타냅니다. `ariaControls` 및 `ariaHasPopup` 예도 확인하는 것이 좋습니다.

      <td>
        `ariaHasPopup` <h5>부울|문자열</h5>
      </td>

      <td>
        이를 사용하여 `Button` 가 대화형 오버레이 요소를 전환함을 나타냅니다. 값은 다음과 같습니다.

        * `false` (기본값)은 요소에 팝업이 없음을 나타냅니다.
        * `true` 팝업에 `menu` 역할이 있음을 나타냅니다.
        * `"menu"` 팝업에 `menu` 역할이 있음을 나타냅니다.
        * `"listbox"` 팝업에 `listbox` 역할이 있음을 나타냅니다.
        * `"tree"` 팝업에 `tree` 역할이 있음을 나타냅니다.
        * `"grid"` 팝업에 `grid` 역할이 있음을 나타냅니다.
        * `"dialog"` 팝업에 `dialog` 역할이 있음을 나타냅니다 [. 이 속성에 대한 WAI ARIA 사양을](https://www.w3.org/TR/wai-aria-1.1/#aria-haspopup) 읽고 어느 것이 사용 사례에 더 적합한지 알아보세요.

        ```js
        const { modalHidden } = this.state;

        return (
          <>
            <Button
              ariaHasPopup="dialog"
              ariaExpanded={!modalHidden}
              onClick={() => this.setState({ modalHidden: false })}
            >
              Open modal
            </Button>
            <Modal
              hidden={modalHidden}
              onClose={() => this.setState({ modalHidden: true })}
            />
          </>
        );
        ```
      </td>
