왼쪽 또는 오른쪽 영역에만 사용되며 축소 기능이 있는 `<Layout>` 구성요소의 하위 요소입니다. "제어되는" 구성 요소로 사용될 수 있습니다(즉, `collapsed` 부울 속성을 전달하여). 또는 기본적으로 자체(비)접힌 상태를 관리합니다. 내장된 트리거를 선택할 수 있는 `triggerType` 소품이 필요합니다. 측면 영역에는 사용자가 영역을 접었다가 다시 확장할 수 있는 버튼이 포함됩니다. 또는 사용자 지정(다른 곳에서 자신의 UI 요소를 사용하여 접힌(비)상태를 제어합니다.

사용법 예시는 [`<Layout>` 문서를](/docs/new-relic-solutions/build-nr-ui/sdk-component/structure/Layout/#examples) 참조하세요.

`<Layout>` 구성요소는 `<CollapsibleLayoutItem>` 및 `<LayoutItem>` 간에 공유되는 각 `type` 의 하위 항목 하나만 지원합니다.

### 용법

```js
import { CollapsibleLayoutItem } from 'nr1'
```
