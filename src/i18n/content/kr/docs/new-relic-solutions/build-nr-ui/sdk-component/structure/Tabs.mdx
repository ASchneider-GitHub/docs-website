탭은 다양한 관련 콘텐츠를 별도의 섹션으로 그룹화하는 데 사용됩니다. 해당 섹션은 화면의 전부이거나 일부에 포함될 수 있습니다. 화면당 하나의 탭 세트를 사용하고 다양한 관련 콘텐츠 유형에만 사용하는 것이 좋습니다. 예를 들어 단일 엔터티에 대한 다양한 데이터가 있는 경우 해당 데이터를 여러 섹션에 버킷할 수 있습니다.

만들기와 같은 작업을 탭의 다른 콘텐츠 섹션과 분리하지 마세요. 작업은 아이콘이나 버튼에 의해 트리거되어야 하며 모달 또는 스택 페이지를 열어야 합니다.

### 용법

```js
import { Tabs } from 'nr1'
```

### 예

#### 기초적인

```js
<Tabs defaultValue="tab-3">
  <TabsItem value="tab-1" label="Tab 1">
    Tab 1 content
  </TabsItem>
  <TabsItem value="tab-2" label="Another tab with a longer name">
    Tab 2 content
  </TabsItem>
  <TabsItem value="tab-3" label="A tab open by default">
    Tab 3 content
  </TabsItem>
</Tabs>
```

#### 콜백 렌더링

```js
function render() {
  const items = new Array(10000).fill().map((_, i) => ({
    value: i,
    content: `Tab ${i} content...`,
    label: `Item ${i}`,
  }));


  return (
    <div className="nr1-Example--tabs">
      <Tabs items={items}>
        {({ item }) => (
          <TabsItem value={item.value} label={item.label}>
            {item.content}
          </TabsItem>
        )}
      </Tabs>
    </div>
  );
}
```
