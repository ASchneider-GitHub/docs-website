스피너는 콘텐츠가 화면에 로드 중임을 사용자에게 알리는 데 사용됩니다.

### 용법

```js
import { Spinner } from 'nr1'
```

### 예

#### 기초적인

```js
<Spinner />
```

#### 점

```js
<Spinner type={Spinner.TYPE.DOT} />
```

#### 인라인

```js
<Spinner inline />
```

### 소품

      </td>
    </tr>

      <td>
        `spacingType` <h5>열거형\[]</h5>
      </td>

      <td>

        <OptionReference array>
          Spinner.SPACING_TYPE.EXTRA_LARGE, Spinner.SPACING_TYPE.LARGE, Spinner.SPACING_TYPE.MEDIUM, Spinner.SPACING_TYPE.NONE, Spinner.SPACING_TYPE.OMIT, Spinner.SPACING_TYPE.SMALL
        </OptionReference>
      </td>
    </tr>

    <tr>
      <td>
        `style` <h5>물체</h5>
      </td>

      <td>
        사용자 정의 스타일을 위한 인라인 스타일입니다. 위치 지정 및 간격 지정 목적으로만 사용해야 합니다.
      </td>
    </tr>

    <tr>
      <td>
        `testId` <h5>끈</h5>
      </td>

      <td>
        `data-test-id` 속성을 추가합니다. 이를 사용하여 단위 및 E2E 테스트의 구성 요소를 목표로 삼습니다. 테스트 ID가 유효하려면 앞에 nerdpack ID를 붙이고 그 뒤에 점이 옵니다. 예를 들어 `my-nerdpack.some-element`. **참고:** `data-test-id` 속성이 DOM 에서 제거될 때 표시되지 않을 수 있습니다. 이를 디버그하려면 `e2e-test` 쿼리 변수를 URL에 전달해야 합니다.
