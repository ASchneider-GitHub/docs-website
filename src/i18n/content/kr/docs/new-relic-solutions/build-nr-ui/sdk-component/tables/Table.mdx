고정된 헤더와 행이 있는 테이블을 렌더링합니다. 테이블은 다음 기능을 구현합니다.

* 유연한 레이아웃: 테이블 헤더는 다양한 크기를 허용하여 유동적이고 고정된 레이아웃을 가능하게 합니다. `TableHeaderCell` 확인하여 열을 맞춤설정하는 방법에 대한 자세한 정보를 찾을 수 있습니다.
* 정렬: 전달된 항목은 테이블의 현재 정렬 상태에 따라 테이블별로 내부적으로 정렬될 수 있습니다. `TableHeaderCell` 확인하면 정렬에 대한 자세한 내용을 확인할 수 있습니다.
* 행 선택: 오른쪽의 확인란을 통해 행을 선택할 수 있습니다. `selected` 소품에서 자세한 정보를 찾을 수 있습니다.
* 행 작업: 상황별 작업이 행별로 트리거될 수 있으므로 사용자는 해당 작업에 대한 기능을 달성할 수 있습니다. `TableRow` 에서 추가하는 방법에 대한 자세한 정보를 찾을 수 있습니다.
* 미리 정의된 사용자 지정 셀: 매우 일반적인 셀 패턴(엔터티 제목, 지표 등)이 이미 플랫폼에서 제공되므로 행의 일부로 반환하기만 하면 됩니다.
* 가상화: 셀은 화면에 표시되는 경우에만 렌더링됩니다. 이를 통해 테이블은 성능 저하가 거의 없이 대규모 데이터 세트와 함께 작동할 수 있습니다.

### 용법

```js
import { Table } from 'nr1'
```

#### 단일 열 정렬

```js
class Example extends React.Component {
  constructor(...args) {
    super(...args);


    this.state = {
      column: 0,
      sortingType: TableHeaderCell.SORTING_TYPE.NONE,
    };


    this.exampleItems = [
      { x: 'A', y: 0 },
      { x: 'B', y: 0 },
      { x: 'A', y: 1 },
    ];
  }


  _onClickTableHeaderCell(column, evt, { nextSortingType }) {
    if (column === this.state.column) {
      this.setState({ sortingType: nextSortingType });
    } else {
      this.setState({ column: column, sortingType: nextSortingType });
    }
  }


  render() {
    const sortingType0 =
      this.state.column === 0
        ? this.state.sortingType
        : TableHeaderCell.SORTING_TYPE.NONE;


    const sortingType1 =
      this.state.column === 1
        ? this.state.sortingType
        : TableHeaderCell.SORTING_TYPE.NONE;


    return (
      <Table items={this.exampleItems}>
        <TableHeader>
          <TableHeaderCell
            sortable
            sortingType={sortingType0}
            onClick={this._onClickTableHeaderCell.bind(this, 0)}
            value={({ item }) => item.x}
          >
            Column 1
          </TableHeaderCell>


          <TableHeaderCell
            sortable
            sortingType={sortingType1}
            onClick={this._onClickTableHeaderCell.bind(this, 1)}
            value={({ item }) => item.y}
          >
            Column 2
          </TableHeaderCell>
        </TableHeader>


        {({ item }) => (
          <TableRow>
            <TableRowCell>{item.x}</TableRowCell>
            <TableRowCell>{item.y}</TableRowCell>
          </TableRow>
        )}
      </Table>
    );
  }
}
```
