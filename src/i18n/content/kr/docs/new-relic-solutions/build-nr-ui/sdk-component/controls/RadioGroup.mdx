`<Radio>` 버튼 그룹입니다. `<Radio>` 버튼은 라디오 그룹의 직계 하위 항목이거나 하위 항목일 수 있습니다. 라디오 그룹 내의 `<Radio>` 버튼에는 고유한 `value` 이 할당되어야 합니다.

라디오 그룹이 설정되면 해당 그룹에서 `<Radio>` 선택하면 그룹에서 현재 선택된 `<Radio>` 이 자동으로 선택 취소됩니다.

**참고**: `value` 설정은 `RadioGroup` 를 제어된 상태로 전환하므로 `defaultValue` 재정의합니다. `value` `checked` 가 설정된 `Radio` 구성요소를 재정의하지 않습니다. 왜냐하면 `RadioGroup` 제어되지 않은 `Radio` 구성요소를 제어하고 제어된 구성요소를 재정의할 수 없기 때문입니다.

`RadioGroup` 에 대한 `onChange` 이벤트 핸들러는 개별 `Radio` 구성요소에 설정된 `onChange` 이벤트 핸들러 이후에 실행됩니다. 그러나 제어된 `RadioGroup` 에서 전체 그룹에 대해 하나의 이벤트 핸들러만 설정하는 것이 좋습니다.

### 용법

```js
import { RadioGroup } from 'nr1'
```

### 예

#### 기초적인

```js
<RadioGroup defaultValue="2">
  <Radio label="Radio 1" value="1" />
  <Radio label="Radio 2" value="2" />
  <Radio label="Radio 3" value="3" />
</RadioGroup>
```

#### 라벨 및 정보 포함

```js
<RadioGroup label="Radio Group" info="Info value" defaultValue="2">
  <Radio label="Radio 1" value="1" />
  <Radio label="Radio 2" value="2" />
  <Radio label="Radio 3" value="3" />
</RadioGroup>
```

#### 인라인 라벨 포함

```js
<RadioGroup label="Radio Group" labelInline defaultValue="2">
  <Radio label="Radio 1" value="1" />
  <Radio label="Radio 2" value="2" />
  <Radio label="Radio 3" value="3" />
</RadioGroup>
```

#### 설명 포함

```js
<RadioGroup
  label="Radio Group"
  description="Description value"
  defaultValue="2"
>
  <Radio label="Radio 1" value="1" />
  <Radio label="Radio 2" value="2" />
  <Radio label="Radio 3" value="3" />
</RadioGroup>
```

#### 잘못된 메시지 포함

```js
<RadioGroup
  label="Radio Group"
  invalid="Invalid message value"
  defaultValue="2"
>
  <Radio label="Radio 1" value="1" />
  <Radio label="Radio 2" value="2" />
  <Radio label="Radio 3" value="3" />
</RadioGroup>
```

#### 제어되는 구성 요소

```js
class MyNerdlet extends React.PureComponent {
  constructor(props) {
    super(props);


    this.state = {
      selectedValue: '2',
    };


    this.onChange = this.onChange.bind(this);
  }


  onChange(event, value) {
    this.setState((state) => {
      return { selectedValue: value };
    });
  }


  render() {
    return (
      <RadioGroup value={this.state.selectedValue} onChange={this.onChange}>
        <Radio label="Radio 1" value="1" />
        <Radio label="Radio 2" value="2" />
        <Radio label="Radio 3" value="3" />
      </RadioGroup>
    );
  }
}
```
