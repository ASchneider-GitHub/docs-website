特定のユーザーを表示する行テーブル セルをレンダリングします。 このセルはユーザー オブジェクトを子として受け取ります。ユーザー オブジェクトは NerdGraph ユーザー オブジェクトと互換性があります (少なくとも`name`フィールドと`gravatar`フィールドが必要です)。

### 使用方法

```js
import { UserTableRowCell } from 'nr1'
```

### 例

#### 例1

```js
function render() {
  const items = [
    {
      name: 'John Doe',
      gravatar:
        'https://secure.gravatar.com/avatar/79988df068cb4d952284a3395412dc4f',
    },
  ];


  return (
    <Table items={items}>
      <TableHeader>
        <TableHeaderCell>User</TableHeaderCell>
      </TableHeader>


      {({ item }) => (
        <TableRow>
          <UserTableRowCell value={item} />
        </TableRow>
      )}
    </Table>
  );
}
```
