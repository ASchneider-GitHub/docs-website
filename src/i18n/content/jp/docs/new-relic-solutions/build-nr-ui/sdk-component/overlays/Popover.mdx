`<Popover>` 必要に応じて小さなオーバーレイを表示するための切り替え可能なオーバーレイです。

`<Popover>`コンポーネントでは、オーバーレイの開閉を処理するトリガーを`<PopoverTrigger>`で提供し、オーバーレイのコンテンツを`<PopoverBody />`経由で提供する必要があります。

### 使用方法

```js
import { Popover } from 'nr1'
```

### 例

#### ベーシック

```js
<Popover>
  <PopoverTrigger>Trigger</PopoverTrigger>
  <PopoverBody>
    <BlockText>Body</BlockText>
  </PopoverBody>
</Popover>
```

#### トリガーとしてのボタン

```js
<Popover>
  <PopoverTrigger>
    <Button>Trigger</Button>
  </PopoverTrigger>
  <PopoverBody>
    <BlockText>Body</BlockText>
  </PopoverBody>
</Popover>
```

#### トリガーとしてのテキストフィールド

```js
<Popover>
  <PopoverTrigger>
    <TextField />
  </PopoverTrigger>
  <PopoverBody>
    <BlockText>Body</BlockText>
  </PopoverBody>
</Popover>
```

#### ポップオーバーフッター付き

```js
<Popover>
  <PopoverTrigger>
    <Button>Trigger</Button>
  </PopoverTrigger>
  <PopoverBody>
    <Card style={{ width: '250px' }}>
      <CardBody>
        <HeadingText>Heading</HeadingText>
        <BlockText
          spacingType={[
            BlockText.SPACING_TYPE.MEDIUM,
            BlockText.SPACING_TYPE.NONE,
          ]}
        >
          Item content
        </BlockText>
        <BlockText>Item content</BlockText>
      </CardBody>
    </Card>
    <PopoverFooter style={{ textAlign: 'right' }}>Info message</PopoverFooter>
  </PopoverBody>
</Popover>
```

#### コントロールされたポップオーバー

```js
class ControlledPopover extends React.Component {
  constructor() {
    super(...arguments);


    this.state = {
      opened: false,
    };


    this._onChange = this._onChange.bind(this);
  }


  _onChange(evt, opened) {
    this.setState({ opened }, () =>
      console.log('controlled state', evt, this.state),
    );
  }


  render() {
    const { opened } = this.state;


    return (
      <Popover opened={opened} onChange={this._onChange}>
        <PopoverTrigger>
          <Button>Trigger</Button>
        </PopoverTrigger>
        <PopoverBody>
          <Card style={{ width: '250px' }}>
            <CardBody>
              <HeadingText>Heading</HeadingText>
              <BlockText
                spacingType={[
                  BlockText.SPACING_TYPE.MEDIUM,
                  BlockText.SPACING_TYPE.NONE,
                ]}
              >
                Item content
              </BlockText>
              <BlockText>Item content</BlockText>
            </CardBody>
          </Card>
          <PopoverFooter style={{ textAlign: 'right' }}>
            Info message
          </PopoverFooter>
        </PopoverBody>
      </Popover>
    );
  }
}
```
