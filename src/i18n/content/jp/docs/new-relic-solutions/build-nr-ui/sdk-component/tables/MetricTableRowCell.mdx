特定のメトリックを表示する行テーブル セルをレンダリングします。 メトリックは渡された単位に従ってフォーマットされます。 単位は、 `<NrqlQuery>`形式タイプ`CHART`でサポートされている単位と 1:1 で互換性があります。

単位と接頭辞の書式は、プラットフォームの他の部分と一致するように自動的に選択されます。 メトリックは自動的に左揃えされ、右から省略されます。これがメトリックを表示するための標準的な方法です。

### 使用方法

```js
import { MetricTableRowCell } from 'nr1'
```

### 例

#### ベーシック

```js
function render() {
  const items = [
    {
      value: 123456789,
    },
  ];


  return (
    <Table items={items}>
      <TableHeader>
        <TableHeaderCell alignmentType={TableHeaderCell.ALIGNMENT_TYPE.RIGHT}>
          Memory usage
        </TableHeaderCell>
      </TableHeader>


      {({ item }) => (
        <TableRow>
          <MetricTableRowCell
            type={MetricTableRowCell.TYPE.BITS}
            value={item.value}
          />
        </TableRow>
      )}
    </Table>
  );
}
```

#### 付加価値付き

```js
function render() {
  const items = [
    {
      value: 123456789,
      lastUpdated: new Date().toLocaleTimeString(),
    },
  ];


  return (
    <Table items={items} multivalue>
      <TableHeader>
        <TableHeaderCell alignmentType={TableHeaderCell.ALIGNMENT_TYPE.RIGHT}>
          Memory usage
        </TableHeaderCell>
      </TableHeader>


      {({ item }) => (
        <TableRow>
          <MetricTableRowCell
            type={MetricTableRowCell.TYPE.BITS}
            value={item.value}
            additionalValue={`Last updated at ${item.lastUpdated}`}
          />
        </TableRow>
      )}
    </Table>
  );
}
```
