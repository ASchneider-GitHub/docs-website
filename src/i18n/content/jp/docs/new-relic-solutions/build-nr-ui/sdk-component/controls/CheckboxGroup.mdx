`<Checkbox>`のボタンのグループ。 `<Checkbox>`ボタンは、チェックボックス グループの直接の子または子孫のいずれかになります。

**注意**: `value`を設定すると、チェックボックスのグループが制御された状態になるため、 `defaultValue`が上書きされます。 `value`は、 `checked`が設定されているチェックボックスを上書きしません。これは、 `CheckboxGroup`が制御されていない`Checkbox`コンポーネントを制御し、制御されているコンポーネントを上書きできないためです。 ただし、 `value` `Checkbox`の`defaultChecked`をオーバーライドします。

`CheckboxGroup`の`onChange`イベント ハンドラーは、個々の`Checkbox`コンポーネントに設定された`onChange`イベント ハンドラーの後に起動されます。 ただし、制御された`CheckboxGroup`内のグループ全体に対して 1 つのイベント ハンドラーのみを設定することを強くお勧めします。

`Checkbox` コンポーネントを`CheckboxGroup`で制御するには、そのコンポーネントに`value`を設定する**必要があります**。

### 使用方法

```js
import { CheckboxGroup } from 'nr1'
```

### 例

#### ベーシック

```js
<CheckboxGroup label="My checkbox group">
  <Checkbox label="Checkbox 1" />
  <Checkbox label="Checkbox 2" />
  <Checkbox label="Checkbox 3" />
</CheckboxGroup>
```

#### ラベルと情報付き

```js
<CheckboxGroup label="My checkbox group" info="Info value">
  <Checkbox label="Checkbox 1" />
  <Checkbox label="Checkbox 2" />
  <Checkbox label="Checkbox 3" />
</CheckboxGroup>
```

#### インラインラベル付き

```js
<CheckboxGroup label="My checkbox group" labelInline>
  <Checkbox label="Checkbox 1" />
  <Checkbox label="Checkbox 2" />
  <Checkbox label="Checkbox 3" />
</CheckboxGroup>
```

#### 説明付き

```js
<CheckboxGroup label="My checkbox group" description="Description value">
  <Checkbox label="Checkbox 1" />
  <Checkbox label="Checkbox 2" />
  <Checkbox label="Checkbox 3" />
</CheckboxGroup>
```

#### 無効なメッセージ

```js
<CheckboxGroup label="My checkbox group" invalid="Invalid message value">
  <Checkbox label="Checkbox 1" />
  <Checkbox label="Checkbox 2" />
  <Checkbox label="Checkbox 3" />
</CheckboxGroup>
```

#### デフォルト値の場合

```js
<CheckboxGroup label="My checkbox group" defaultValue={['1', '2']}>
  <Checkbox label="Checkbox 1" value="1" />
  <Checkbox label="Checkbox 2" value="2" />
  <Checkbox label="Checkbox 3" value="3" />
</CheckboxGroup>
```

#### 制御対象コンポーネント

```js
class MyNerdlet extends React.PureComponent {
  constructor(props) {
    super(props);


    this.state = {
      values: ['2'],
    };


    this.onChange = this.onChange.bind(this);
  }


  onChange(event, values) {
    this.setState({ values });
  }


  render() {
    return (
      <CheckboxGroup value={this.state.values} onChange={this.onChange}>
        <Checkbox label="Checkbox 1" value="1" />
        <Checkbox label="Checkbox 2" value="2" />
        <Checkbox label="Checkbox 3" value="3" />
      </CheckboxGroup>
    );
  }
}
```
