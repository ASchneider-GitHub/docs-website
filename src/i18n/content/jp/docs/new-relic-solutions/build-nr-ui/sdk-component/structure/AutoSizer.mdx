`<AutoSizer>`コンポーネントは、それが含まれる要素のサイズの変化を測定してそれに反応する機能を提供します。 外側の要素の`width`または`height`が変更されると、子要素が再レンダリングされ、新しい測定値がオブジェクトとして受け取られます。

### 使用方法

```js
import { AutoSizer } from 'nr1'
```

### 例

#### ベーシック

```js
function render() {
  const style = {
    backgroundColor: '#d8211a1a',
    border: 'dashed 1px #d8211a4d',
  };


  return (
    <AutoSizer>
      {({ width, height }) => (
        <div style={{ ...style, height, width }}>
          width: {width}, height: {height}
        </div>
      )}
    </AutoSizer>
  );
}
```
