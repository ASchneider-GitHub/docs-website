`<Layout>`コンポーネントの子要素。左側または右側の領域にのみ使用され、折りたたむことができます。 「制御された」コンポーネントとして使用できます（つまり、 `collapsed`ブール値プロパティを渡すことによって）、またはデフォルトでは独自の折りたたみ状態（または折りたたみ解除状態）を管理します。 `triggerType`プロパティを使用すると、組み込みトリガーを選択できます。サイド領域には、ユーザーが領域を折りたたんでから再度展開できるようにするボタンが含まれます。またはカスタム、つまり、他の場所で独自の UI 要素を使用して折りたたみ状態 (または展開状態) を制御することもできます。

使用例については[、 `<Layout>`ドキュメントを](/docs/new-relic-solutions/build-nr-ui/sdk-component/structure/Layout/#examples)参照してください。

`<Layout>`コンポーネントは、 `<CollapsibleLayoutItem>`と`<LayoutItem>`間で共有される各`type`の 1 つの子のみをサポートします。

### 使用方法

```js
import { CollapsibleLayoutItem } from 'nr1'
```
