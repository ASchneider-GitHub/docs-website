すべてのアカウントを取得し、ドロップダウンと検索を通じてアカウントを選択できるようにします。 現在選択されているアカウントをキャプチャするには、 `onChange`リスナーを使用できます。

プラットフォームにはすでに独自のアカウント ピッカーが含まれており、そのヘッダーに配置されていることに注意してください。 これは、 `nerdlet.setConfig` API ( `{ accountPicker: true }`を渡す) を使用して表示するように構成でき、その値は`<PlatformStateContext.Consumer>`を使用して読み取ることができます。 詳しい情報については、これらの API を確認してください。

### 使用方法

```js
import { AccountPicker } from 'nr1'
```

### 例

#### ベーシック

```js
class Example extends React.Component {
  constructor() {
    super(...arguments);


    this.state = { accountId: null };


    this.onChangeAccount = this.onChangeAccount.bind(this);
  }


  onChangeAccount(_, value) {
    alert(`Selected account: ${value}`);


    this.setState({ accountId: value });
  }


  render() {
    return (
      <AccountPicker
        value={this.state.accountId}
        onChange={this.onChangeAccount}
      />
    );
  }
}
```

#### インラインラベル付き

```js
class Example extends React.Component {
  constructor() {
    super(...arguments);


    this.state = { accountId: null };


    this.onChangeAccount = this.onChangeAccount.bind(this);
  }


  onChangeAccount(_, value) {
    alert(`Selected account: ${value}`);


    this.setState({ accountId: value });
  }


  render() {
    return (
      <AccountPicker
        label="Account"
        labelInline
        value={this.state.accountId}
        onChange={this.onChangeAccount}
      />
    );
  }
}
```
