ツールチップは、UI ではすぐには利用できない追加情報を表示するために使用されます。 これらを使用して、UI 要素が何であるかを説明したり、アクションの目的を説明したり、単語やフレーズの定義を提供したりできます。

### 使用方法

```js
import { Tooltip } from 'nr1'
```

### 例

#### ベーシック

```js
<Tooltip text="Search across all your accounts">
  <Button>Search</Button>
</Tooltip>
```

#### 配置タイプあり

```js
<Tooltip
  text="Explore your metrics, events and logs"
  placementType={Tooltip.PLACEMENT_TYPE.BOTTOM}
>
  <Button>Query your data</Button>
</Tooltip>
```

#### 追加情報リンク付き

```js
function render() {
  const additionalInfoLink = {
    label: 'See the docs',
    to: 'https://docs.newrelic.com',
  };


  return (
    <p>
      Welcome to
      <Tooltip
        text="Go to docs for more information"
        placementType={Tooltip.PLACEMENT_TYPE.BOTTOM}
        additionalInfoLink={additionalInfoLink}
      >
        New Relic One
      </Tooltip>
    </p>
  );
}
```

        * コピーは読みやすく、明確なものにしてください。 長いコピーの場合は、完全な文、能動態の動詞、句読点を使用します。
        * 誤解を避けるために十分な情報を提供してください。 たとえば、ボタンが複数の機能を追加できるように見える場合は、ツールチップを使用して、実際に何が追加されるかを明確にします。
