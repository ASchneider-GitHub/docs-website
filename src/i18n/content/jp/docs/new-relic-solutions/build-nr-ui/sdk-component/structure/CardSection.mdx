`<CardBody>`コンポーネントの子要素。

### 使用方法

```js
import { CardSection } from 'nr1'
```

### 小道具

<table>
  <tbody>
    <tr>
      <td>
        `children` <h5>必須</h5><h5>ノード</h5>
      </td>

      <td>
        カード内にレンダリングするコンテンツ。
      </td>
    </tr>

    <tr>
      <td>
        `className` <h5>ストリング</h5>
      </td>

      <td>
        コンポーネントにクラス名を追加します。配置と間隔の目的にのみ使用してください。
      </td>
    </tr>

    <tr>
      <td>
        `collapsed` <h5>ブール値</h5>
      </td>

      <td>
        コンポーネントの折りたたまれた状態を制御するには、このプロパティを設定します。定義されている場合、コンポーネントは[制御されたコンポーネント](https://facebook.github.io/react/docs/forms.html)になります。
      </td>
    </tr>

    <tr>
      <td>
        `collapsible` <h5>ブール値</h5>
      </td>

      <td>
        コンポーネントを折りたたみ可能にするには、このプロパティを設定します。
      </td>
    </tr>

    <tr>
      <td>
        `defaultCollapsed` <h5>ブール値</h5>
      </td>

      <td>
        `true`の場合、カードの初期の折りたたまれた状態は`collapsed`です。[制御されたコンポーネント](https://facebook.github.io/react/docs/forms.html)を使用しない場合に便利です。
      </td>
    </tr>

    <tr>
      <td>
        `onChange` <h5>機能</h5>
      </td>

      <td>
        この関数は、ユーザーがヘッダーをクリックしてコンポーネントを折りたたんだり展開したりするたびに呼び出されます。 コンポーネントの状態を制御する場合は、これを使用して`collapsed`プロパティの値を設定します。

        <FunctionDefinition
          returnValue={[]}
          arguments={[{"name":"event","type":"React.MouseEvent","description":""},{"name":"collapsed","type":"boolean","description":""}]}
        />
      </td>
    </tr>

    <tr>
      <td>
        `style` <h5>物体</h5>
      </td>

      <td>
        カスタム スタイル設定用のインライン スタイル。配置と間隔設定の目的でのみ使用してください。
      </td>
    </tr>

    <tr>
      <td>
        `testId` <h5>ストリング</h5>
      </td>

      <td>
        `data-test-id`属性を追加します。 ユニット テストと E2E テストでコンポーネントをターゲットにするために使用します。テスト ID を有効にするには、nerdpack ID をプレフィックスとして付け、その後にドットを付けます。たとえば、 `my-nerdpack.some-element` 。

        **注:** `data-test-id` 属性はDOMから削除されているため表示されない場合があります。デバッグするには、URL に `e2e-test` クエリ引数を渡します。
      </td>
    </tr>
  </tbody>
</table>
