タブは、さまざまな関連コンテンツを個別のセクションにグループ化するのに役立ちます。 これらのセクションは、画面のすべて、または一部に含まれていてもかまいません。 画面ごとに 1 セットのタブを使用し、関連するさまざまなコンテンツ タイプにのみ使用することをお勧めします。 たとえば、単一のエンティティに関する異なるデータがある場合、データを異なるセクションに分割することができます。

作成などのアクションをタブ内の他のコンテンツ セクションから分離しないでください。 アクションはアイコンまたはボタンによってトリガーされ、モーダル ページまたはスタック ページのいずれかが開かれます。

### 使用方法

```js
import { Tabs } from 'nr1'
```

### 例

#### ベーシック

```js
<Tabs defaultValue="tab-3">
  <TabsItem value="tab-1" label="Tab 1">
    Tab 1 content
  </TabsItem>
  <TabsItem value="tab-2" label="Another tab with a longer name">
    Tab 2 content
  </TabsItem>
  <TabsItem value="tab-3" label="A tab open by default">
    Tab 3 content
  </TabsItem>
</Tabs>
```

#### レンダリングコールバック

```js
function render() {
  const items = new Array(10000).fill().map((_, i) => ({
    value: i,
    content: `Tab ${i} content...`,
    label: `Item ${i}`,
  }));


  return (
    <div className="nr1-Example--tabs">
      <Tabs items={items}>
        {({ item }) => (
          <TabsItem value={item.value} label={item.label}>
            {item.content}
          </TabsItem>
        )}
      </Tabs>
    </div>
  );
}
```
