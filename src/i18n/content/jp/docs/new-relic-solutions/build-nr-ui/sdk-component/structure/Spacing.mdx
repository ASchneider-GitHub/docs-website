ラップされたコンポーネントに間隔を提供します。 間隔は、0、1、2、3、または 4 つの値を含むことができるタプルとして提供されます。 これらの各値は、TYPE 静的プロパティから取得されます。

タプルの構造は、 `margin`または`padding`プロパティの動作と非常に似ています。1 つの値を指定すると、その値が 4 つの側面に複製され、2 つの値を指定すると、 `top` / `bottom`と`left` / `right`に複製されます。 特定の値を省略して他の値を渡す場合は、プレースホルダーとして`OMIT`を使用できます。 使用される側のスペースはそのまま残されます。

SDK の他のコンポーネントには、間隔調整機能が組み込まれています。 独自のコンポーネントで同じ機能を複製する場合は、 `Spacing.add`使用してコンポーネントを渡します。メソッドによって返されるコンポーネントには、 `type`プロパティと静的プロパティが含まれます。

### 使用方法

```js
import { Spacing } from 'nr1'
```
