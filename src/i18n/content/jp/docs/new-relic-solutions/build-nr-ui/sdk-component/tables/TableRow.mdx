単一の行を構成するセルのラッパー。

### 使用方法

```js
import { TableRow } from 'nr1'
```

### 小道具

<table>
  <tbody>
    <tr>
      <td>
        `actions` <h5>形\[]</h5>
      </td>

      <td>
        ユーザーが行にマウスを置いたときに表示される一連のアクションを定義します。 アクションには必須のテキストと onClick コールバック (アクションが属する行の項目とインデックスを使用して呼び出されます) があり、disabled プロパティも受け入れます。

        <h3>形</h3>

        `disabled` <h5>ブール値</h5>

        `label` <h5>必須の</h5><h5>文字列。</h5>行のアクションを説明するために使用します。

        `onClick` ユーザーがアクションをクリックするたびに呼び出される<h5>必須</h5><h5>関数</h5>コールバック。

        <FunctionDefinition
          returnValue={[]}
          arguments={[{"name":"event","type":"React.MouseEvent","description":""}]}
        />

        `testId` <h5>文字列</h5>`data-test-id`属性を追加します。 ユニット テストと E2E テストでコンポーネントをターゲットにするために使用します。テスト ID を有効にするには、nerdpack ID をプレフィックスとして付け、その後にドットを付けます。たとえば、 `my-nerdpack.some-element` 。 **注:** `data-test-id` 属性はDOMから削除されているため表示されない場合があります。デバッグするには、URL に `e2e-test` クエリ引数を渡します。

        `type` <h5>enum</h5>アクション ドロップダウン内の各項目のアクション タイプ。必須ではありません。デフォルトは`TableRow.ACTION_TYPE.NORMAL`です。

        * 通常 - 一般的な非クリティカルなアクション。

        * 破壊的 - 削除や除去などの破壊的なアクションがあり、完了する前にユーザーに一時停止して検討してもらいたい場合に使用します。

          <OptionReference>
            TableRow.ACTION_TYPE.DESTRUCTIVE、TableRow.ACTION_TYPE.NORMAL、
          </OptionReference>

        ```js
        const actions = [
          {
            label: 'Duplicate',
            type: TableRow.ACTION_TYPE.NORMAL,
          },
          {
            label: 'Delete',
            type: TableRow.ACTION_TYPE.DESTRUCTIVE,
          },
        ];
        ```

        ```js
        const actions = [
          {
            label: 'Action 1',
            disabled: true,
            onClick: (evt, { item, index }) => {
              alert(`Action 1:Item: ${index}${JSON.stringify(item, null, 2)}`);
            },
          },
          {
            label: 'Action 2',
            onClick: (evt, { item, index }) => {
              alert(`Action 2:Item: ${index}${JSON.stringify(item, null, 2)}`);
            },
          },
        ];
        ```
