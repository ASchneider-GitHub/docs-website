`<Popover>`コンポーネントの子要素。

ポップオーバーオーバーレイのコンテンツが含まれます。

### 使用方法

```js
import { PopoverBody } from 'nr1'
```

### 小道具

<table>
  <tbody>
    <tr>
      <td>
        `children` <h5>必須</h5><h5>ノード</h5>
      </td>

      <td>
        ポップオーバーが開いたときにレンダリングされるコンテンツ。
      </td>
    </tr>

    <tr>
      <td>
        `className` <h5>ストリング</h5>
      </td>

      <td>
        コンポーネントにクラス名を追加します。配置と間隔の目的にのみ使用してください。
      </td>
    </tr>

    <tr>
      <td>
        `onClose` <h5>機能</h5>
      </td>

      <td>
        `Popover`が制御されていないときにオーバーレイが閉じられるたびにコールバックが起動されます。 `Popover`が制御されている場合は発動しません。

        <FunctionDefinition
          returnValue={[]}
          arguments={[{"name":"event","type":"React.MouseEvent","description":""}]}
        />
      </td>
    </tr>

    <tr>
      <td>
        `onOpen` <h5>機能</h5>
      </td>

      <td>
        `Popover`が制御されていないときにオーバーレイが開かれるたびにコールバックが起動されます。 `Popover`が制御されている場合は発動しません。

        <FunctionDefinition
          returnValue={[]}
          arguments={[{"name":"event","type":"React.MouseEvent","description":""}]}
        />
      </td>
    </tr>

    <tr>
      <td>
        `onToggle` <h5>機能</h5>
      </td>

      <td>
        `Popover`が制御されていないときにオーバーレイが切り替えられるたびにコールバックが起動されます。 `Popover`が制御されている場合は発動しません。

        <FunctionDefinition
          returnValue={[]}
          arguments={[{"name":"event","type":"React.MouseEvent","description":""},{"name":"opened","type":"boolean","description":""}]}
        />
      </td>
    </tr>

    <tr>
      <td>
        `placementType` <h5>enum</h5>
      </td>

      <td>
        ポップオーバー トリガーに対するポップオーバー本体の配置を指定します。 デフォルトでは下から表示しますが、下端から表示するように選択することもできます。

        <OptionReference>
          PopoverBody.PLACEMENT_TYPE.BOTTOM_END、PopoverBody.PLACEMENT_TYPE.BOTTOM_START
        </OptionReference>
      </td>
    </tr>

    <tr>
      <td>
        `style` <h5>物体</h5>
      </td>

      <td>
        カスタム スタイル設定用のインライン スタイル。配置と間隔設定の目的でのみ使用してください。
      </td>
    </tr>

    <tr>
      <td>
        `testId` <h5>ストリング</h5>
      </td>

      <td>
        `data-test-id`属性を追加します。 ユニット テストと E2E テストでコンポーネントをターゲットにするために使用します。テスト ID を有効にするには、nerdpack ID をプレフィックスとして付け、その後にドットを付けます。たとえば、 `my-nerdpack.some-element` 。

        **注:** `data-test-id` 属性はDOMから削除されているため表示されない場合があります。デバッグするには、URL に `e2e-test` クエリ引数を渡します。
      </td>
    </tr>
  </tbody>
</table>
