ボタンは、ユーザーにアクションを完了する方法を提供するために、製品全体で使用されます。 体験中にボタンを使いすぎないようにしてください。 これにより、ユーザーはどのようなアクションを取ってほしいかを正確に把握できるようになります。 エクスペリエンスでは、プライマリ ボタンを 1 つ以上使用しないでください。

主な機能 — ボタンは

* 一つのアクションを実行する
* 複数のボタンがある場合は、複数のアクションを実行します。

### 使用方法

```js
import { Button } from 'nr1'
```

### 例

#### ベーシック

```js
<div className="nr1-Docs-prettify">
  <Button onClick={() => alert('Hello World!')} type={Button.TYPE.PRIMARY}>
    Click me
  </Button>
  <Button
    onClick={() => alert('Hello World!')}
    type={Button.TYPE.PRIMARY}
    iconType={
Button.ICON_TYPE.DOCUMENTS__DOCUMENTS__NOTES__A_ADD}
  >
    Click me
  </Button>
  <Button
    type={Button.TYPE.PRIMARY}
    iconType={
Button.ICON_TYPE.DOCUMENTS__DOCUMENTS__FILE__A_ADD}
  />
</div>
```

```js
<div className="nr1-Docs-prettify">
```

### 小道具

<table>
  <tbody>
    <tr>
      <td>
        `ariaControls` <h5>ストリング</h5>
      </td>

      <td>
        パネルを展開または開くために使用されるときに、 `Button`が制御する要素の`id`文字列を渡します。 `ariaExpanded`と一緒に使用してください。

        ```js
        const { expanded } = this.state;
        const panelId = 'content-panel';

        return (
          <>
            <Button ariaControls={panelId} ariaExpanded={expanded}>
              Expand content
            </Button>
            <div id={panelId} hidden={!expanded}>
              Content
            </div>
          </>
        );
        ```
      </td>
    </tr>

    <tr>
      <td>
        `ariaExpanded` <h5>ブール値</h5>
      </td>

      <td>
        `Button`が制御する要素が展開されているかどうかを示すために使用します。 `ariaControls`と`ariaHasPopup`例も確認することをお勧めします。
      </td>
    </tr>

    <tr>
      <td>
        `ariaHasPopup` <h5>ブール値|文字列</h5>
      </td>

      <td>
        `Button`がインタラクティブなオーバーレイ要素を切り替えることを示すために使用します。値は次のとおりです。

        * `false` (デフォルト) 要素にポップアップがないことを示します。
        * `true` ポップアップに`menu`役割があることを示します
        * `"menu"` ポップアップに`menu`役割があることを示します
        * `"listbox"` ポップアップに`listbox`役割があることを示します
        * `"tree"` ポップアップに`tree`役割があることを示します
        * `"grid"` ポップアップに`grid`役割があることを示します
        * `"dialog"` ポップアップに`dialog`ロールがあることを示します[。この属性の WAI ARIA 仕様を](https://www.w3.org/TR/wai-aria-1.1/#aria-haspopup)読んで、どの属性が使用ケースに適しているかを確認してください。

        ```js
        const { modalHidden } = this.state;

        return (
          <>
            <Button
              ariaHasPopup="dialog"
              ariaExpanded={!modalHidden}
              onClick={() => this.setState({ modalHidden: false })}
            >
              Open modal
            </Button>
            <Modal
              hidden={modalHidden}
              onClose={() => this.setState({ modalHidden: true })}
            />
          </>
        );
        ```
      </td>
    </tr>

    <tr>
      <td>
        `ariaLabel` <h5>ストリング</h5>

        ```js
        <Button
          iconType={Button.ICON_TYPE.INTERFACE__OPERATIONS__SEARCH}
          ariaLabel="Search for entities"
        />;
        ```
      </td>
    </tr>
