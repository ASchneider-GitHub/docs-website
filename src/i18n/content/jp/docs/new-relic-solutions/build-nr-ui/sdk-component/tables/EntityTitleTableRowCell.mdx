特定のエンティティのタイトルを表示する行テーブル セルをレンダリングします。 このセルはエンティティ オブジェクトを子として受け取ります。エンティティ オブジェクトは NerdGraph エンティティ オブジェクトと互換性があります (少なくとも`name`フィールドと`reporting`フィールドが必要です)。 アラート可能な場合は、 `alertSeverity`もクエリする必要があります。

### 使用方法

```js
import { EntityTitleTableRowCell } from 'nr1'
```

### 例

#### ベーシック

```js
function render() {
  const items = [
    {
      name: 'Login service',
      guid: 'MTIzNDU2fEZPT3xCQVJ8OTg3NjU0MzIx',
      alertSeverity: 'CRITICAL',
      reporting: true,
    },
  ];


  return (
    <Table items={items}>
      <TableHeader>
        <TableHeaderCell>Entity</TableHeaderCell>
      </TableHeader>


      {({ item }) => (
        <TableRow>
          <EntityTitleTableRowCell value={item} />
        </TableRow>
      )}
    </Table>
  );
}
```

#### 付加価値付き

```js
function render() {
  const items = [
    {
      name: 'Login service',
      guid: 'MTIzNDU2fEZPT3xCQVJ8OTg3NjU0MzIx',
      alertSeverity: 'CRITICAL',
      reporting: true,
    },
  ];


  return (
    <Table items={items} multivalue>
      <TableHeader>
        <TableHeaderCell>Entity</TableHeaderCell>
      </TableHeader>


      {({ item }) => (
        <TableRow>
          <EntityTitleTableRowCell value={item} additionalValue={item.guid} />
        </TableRow>
      )}
    </Table>
  );
}
```
