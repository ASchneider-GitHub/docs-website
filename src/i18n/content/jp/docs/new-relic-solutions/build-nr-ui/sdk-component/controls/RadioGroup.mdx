`<Radio>`のボタンのグループ。 `<Radio>`ボタンは、ラジオ グループの直接の子または子孫のいずれかになります。 ラジオ グループ内の`<Radio>`ボタンには、一意の`value`を割り当てる必要があります。

ラジオ グループが確立されると、そのグループ内の任意の`<Radio>`を選択すると、グループ内で現在選択されているすべての`<Radio>`の選択が自動的に解除されます。

**注意**: `value`を設定すると、 `RadioGroup`が制御された状態になるため、 `defaultValue`が上書きされます。 `value`は、 `checked`が設定されている`Radio`コンポーネントをオーバーライドしません。これは、 `RadioGroup`が制御されていない`Radio`コンポーネントを制御し、制御されているコンポーネントをオーバーライドできないためです。

`RadioGroup`の`onChange`イベント ハンドラーは、個々の`Radio`コンポーネントに設定された`onChange`イベント ハンドラーの後に起動されます。 ただし、制御された`RadioGroup`内のグループ全体に対して 1 つのイベント ハンドラーのみを設定することを強くお勧めします。

### 使用方法

```js
import { RadioGroup } from 'nr1'
```

### 例

#### ベーシック

```js
<RadioGroup defaultValue="2">
  <Radio label="Radio 1" value="1" />
  <Radio label="Radio 2" value="2" />
  <Radio label="Radio 3" value="3" />
</RadioGroup>
```

#### ラベルと情報付き

```js
<RadioGroup label="Radio Group" info="Info value" defaultValue="2">
  <Radio label="Radio 1" value="1" />
  <Radio label="Radio 2" value="2" />
  <Radio label="Radio 3" value="3" />
</RadioGroup>
```

#### インラインラベル付き

```js
<RadioGroup label="Radio Group" labelInline defaultValue="2">
  <Radio label="Radio 1" value="1" />
  <Radio label="Radio 2" value="2" />
  <Radio label="Radio 3" value="3" />
</RadioGroup>
```

#### 説明付き

```js
<RadioGroup
  label="Radio Group"
  description="Description value"
  defaultValue="2"
>
  <Radio label="Radio 1" value="1" />
  <Radio label="Radio 2" value="2" />
  <Radio label="Radio 3" value="3" />
</RadioGroup>
```

#### 無効なメッセージ

```js
<RadioGroup
  label="Radio Group"
  invalid="Invalid message value"
  defaultValue="2"
>
  <Radio label="Radio 1" value="1" />
  <Radio label="Radio 2" value="2" />
  <Radio label="Radio 3" value="3" />
</RadioGroup>
```

#### 制御対象コンポーネント

```js
class MyNerdlet extends React.PureComponent {
  constructor(props) {
    super(props);


    this.state = {
      selectedValue: '2',
    };


    this.onChange = this.onChange.bind(this);
  }


  onChange(event, value) {
    this.setState((state) => {
      return { selectedValue: value };
    });
  }


  render() {
    return (
      <RadioGroup value={this.state.selectedValue} onChange={this.onChange}>
        <Radio label="Radio 1" value="1" />
        <Radio label="Radio 2" value="2" />
        <Radio label="Radio 3" value="3" />
      </RadioGroup>
    );
  }
}
```
