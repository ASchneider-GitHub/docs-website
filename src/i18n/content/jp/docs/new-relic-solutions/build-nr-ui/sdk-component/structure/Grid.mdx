グリッドは 2 次元レイアウト システムです。つまり、列と行の両方を処理できるため、主に 1 次元システムである`<Stack>`とは異なり、グリッドベースのユーザー インターフェースを設計するのに役立ちます。

### 使用方法

```js
import { Grid } from 'nr1'
```
