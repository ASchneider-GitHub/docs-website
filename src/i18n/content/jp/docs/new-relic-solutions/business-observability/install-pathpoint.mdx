New Relic Pathpoint は、ほぼすべての New Relic アカウントと統合して、包括的なビジネス ジャーニーのオブザーバビリティを提供できます。開始するには、組織に関連するテレメトリ データを用意することが重要です。これには、メトリック、イベント、ログ、またはトレースを含めることができます。APM イベントとログは Pathpoint の一般的な出発点ですが、テレメトリ データの選択は完全にユーザー次第です。

Pathpoint を使い始める前に、New Relic アカウントにインストールする必要があります。

## New Relic カタログから Pathpoint をインストールする [#install-catalog]

パスポイントをインストールするには:

<Steps>
  <Step>
    <DNT>**\[one.newrelic.com]\[https&#x3A;//one.newrelic.com] > + Integrations & Agents**</DNT>に移動します。
  </Step>

  <Step>
    <DNT>**Apps and visualizations**</DNT>を選択し、 `pathpoint`を検索します。
  </Step>

  <Step>
    確認済みの<DNT>**Pathpoint**</DNT>のインストレーション アイコンをクリックし、次に<DNT>**Add this app**</DNT>をクリックします。

    <img
      title="Install Pathpoint"
      alt="Install Pathpoint"
      src={installPathpoint}
    />

    Pathpoint が New Relic アカウントに追加され、 <DNT>**Apps**</DNT>からアクセスできるようになりました。
  </Step>
</Steps>

Pathpoint をインストールしたら、変更した JSON ファイルを Pathpoint にアップロードするだけで、その構成をカスタマイズできます。さらに、ステージ、ステップ、およびタッチポイントを変更し、現在の構成を JSON ファイルとして保存できます。[Pathpoint の構成の詳細については、こちらをご覧ください](/docs/business-observability/configure-pathpoint/)。
