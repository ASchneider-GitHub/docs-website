ブラウザー API を使用すると、ブラウザーの監視をカスタマイズおよび拡張できます。API の使用を開始するには、[監視するすべてのページにブラウザ エージェントが存在することを](/docs/browser/new-relic-browser/installation/install-new-relic-browser-agent)確認します。

以下は、各ブラウザ API メソッドの使用例です。

## トレースとイベント [#api-traces]

トレースやイベントを監視するには、次のような方法があります。

<table>
  <thead>
    <tr>
      <th style={{ width: "350px" }}>
        あなたがしたい場合は...
      </th>

      <th>
        これを使用
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        New Relic が自動的に追跡しないアクションを追跡する
      </td>

      <td>
        [newrelic.addPageAction()](/docs/browser/new-relic-browser/browser-apis/addpageaction)を使用する
      </td>
    </tr>

    <tr>
      <td>
        進行中の [セッショントレースにオブジェクトを追加する。](/docs/browser/new-relic-browser/browser-pro-features/session-traces-exploring-webpages-life-cycle)
      </td>

      <td>
        [newrelic.addToTrace()](/docs/browser/new-relic-browser/browser-apis/addtotrace)を使用する
      </td>
    </tr>

    <tr>
      <td>
        独自の基準でページが終了した時点を記録する
      </td>

      <td>
        [newrelic.finished()](/docs/browser/new-relic-browser/browser-apis/finished)を使用する
      </td>
    </tr>

    <tr>
      <td>
        ページ上の後続のイベントに属性名と値を追加する
      </td>

      <td>
        [newrelic.setCustomAttribute()](/docs/browser/new-relic-browser/browser-apis/setcustomattribute)を使用する
      </td>
    </tr>

    <tr>
      <td>
        ページ上の後続のイベントにアプリケーションのバージョン情報を追加します。
      </td>

      <td>
        [newrelic.setApplicationVersion()](/docs/browser/new-relic-browser/browser-apis/setApplicationVersion)を使用する
      </td>
    </tr>

    <tr>
      <td>
        ページ名を設定してページビューをグループ化
      </td>

      <td>
        [newrelic.setPageViewName()](/docs/browser/new-relic-browser/browser-apis/setpageviewname)を使用する
      </td>
    </tr>

    <tr>
      <td>
        ページ上の後続のイベントにユーザー ID 情報を追加します。
      </td>

      <td>
        [newrelic.setUserId()](/docs/browser/new-relic-browser/browser-apis/setuserid)を使用する
      </td>
    </tr>
  </tbody>
</table>

## エラー報告 [#api-errors]

アプリケーションのエラーを報告するには、以下の方法があります。

<table>
  <thead>
    <tr>
      <th style={{ width: "350" }}>

      </th>

    <tr>

    <tr>

      </td>
    </tr>

    <tr>

        SPAのサブコンポーネントのトレース [インタラクション](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction) 個別に

      </td>
    </tr>

    <tr>
      <td>
        [インタラクションを無視する](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction)
      </td>

      <td>
        [newrelic.interaction().ignore()](/docs/browser/new-relic-browser/browser-apis/ignore)を使用する
      </td>
    </tr>

    <tr>
      <td>
        現在の [のインタラクションにバインドされた新しい API オブジェクトを返します。](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction)
      </td>

      <td>
        [newrelic.interaction()](/docs/browser/new-relic-browser/browser-apis/interaction)を使用する
      </td>
    </tr>

    <tr>
      <td>
        [インタラクション](/attribute-dictionary/?event=BrowserInteraction) が記録される前に、名前や無視された属性などの属性を調整する。
      </td>

      <td>
        [newrelic.interaction().onEnd()](/docs/browser/new-relic-browser/browser-apis/onend)を使用する
      </td>
    </tr>

    <tr>
      <td>
        ブラウザ [インタラクション](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction) が終了したときに確実に保存されるようにする。
      </td>

      <td>
        [newrelic.interaction().save()](/docs/browser/new-relic-browser/browser-apis/save)を使用する
      </td>
    </tr>

    <tr>
      <td>
        現在の [インタラクションにカスタム属性を追加する](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction)
      </td>

      <td>
        [newrelic.interaction().setAttribute()](/docs/browser/new-relic-browser/browser-apis/setattribute)を使用します
      </td>
    </tr>

    <tr>
      <td>
        SPAルートにデフォルトの名前よりも正確な名前をつける
      </td>

      <td>
        [newrelic.interaction().setCurrentRouteName](/docs/browser/new-relic-browser/browser-apis/setcurrentroutename)を使用します
      </td>
