<Callout variant="important" title="EOL通知">
  パートナーシップ API は 2023 年 3 月にサポート終了 (EOL) となり、サポートされなくなり、使用できなくなります。
</Callout>

当社のパートナーシップ API により、New Relic パートナーと[顧客パートナーシップ](/docs/new-relic-partnerships/partner-integration-guide/partner-account-maintenance/partnership-accounts-users-subscriptions/#partnerships)として設定された New Relic アカウントは、アカウント、ユーザー、サブスクリプション関連の設定を管理できるようになりました。

Partnership API は利用できなくなったため、Nerdgraph API を使用して多くの同じタスクを実行できます。

* [アカウントの管理](/docs/apis/nerdgraph/examples/manage-accounts-nerdgraph)
* [ユーザーの管理](/docs/apis/nerdgraph/examples/nerdgraph-manage-users)
* [ユーザーグループの管理](/docs/apis/nerdgraph/examples/nerdgraph-manage-groups)
