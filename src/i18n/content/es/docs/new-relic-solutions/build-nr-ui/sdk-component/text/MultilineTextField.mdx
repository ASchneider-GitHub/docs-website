Componente empleado para fines de entrada de varias líneas en formularios.

### Uso

```js
import { MultilineTextField } from 'nr1'
```

### Ejemplos

#### Básico

```js
<MultilineTextField label="Summary" placeholder="Enter a short description" />
```

#### Con etiqueta en línea

```js
<MultilineTextField
  label="Summary"
  labelInline
  placeholder="Enter a short description"
/>
```

#### Preformateado

```js
<MultilineTextField
  type={MultilineTextField.TYPE.PREFORMATTED}
  label="Build command"
  placeholder="npm run build"
/>
```

#### Cargando

```js
<MultilineTextField loading placeholder="Enter a short description" />
```

#### Desactivado

```js
<MultilineTextField disabled placeholder="Enter a short description" />
```

#### Solo lectura

```js
<MultilineTextField
  readOnly
  defaultValue="Some value"
  placeholder="Enter a short description"
/>
```

#### Con descripción

```js
<MultilineTextField
  description="Description value"
  placeholder="Enter a short description"
/>
```

#### Con mensaje inválido

```js
<MultilineTextField
  invalid="Not enough text, needs more"
  placeholder="Enter a short description"
/>
```
