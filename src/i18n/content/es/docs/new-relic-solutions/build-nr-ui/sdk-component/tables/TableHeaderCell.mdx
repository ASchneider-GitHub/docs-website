Envuelve el contenido de una celda de la tabla ubicada en el encabezado de la tabla, manteniéndola en la parte superior y agregando controles adicionales según sea necesario, por ejemplo mecanismos de clasificación.

### Uso

```js
import { TableHeaderCell } from 'nr1'
```

      <td>
        `sortingFunction` <h5>función</h5>
      </td>

      <td>
        Método para ordenar filas. El método predeterminado generalmente hará lo correcto (orden alfabético para texto, orden numérico para números y agrupación booleana), pero si es necesario, esto puede anular. La función toma dos parámetros (`a` y `b`) y tiene que devolver un número negativo si se considera que `a` aparece antes de `b`, uno positivo si se considera que aparece después y cero si ambos valores son idénticos. Cada uno de los objetos pasados a la devolución de llamada contiene el valor del determinado fila (es decir, el resultado de llamar a `value` sobre la fila), el elemento de la fila (como `item`), el índice de la fila (como `index`) y todos los elementos de la matriz `items` (como `items`). Generalmente comparará filas usando `a.item` y `b.item`.

        <FunctionDefinition
          returnValue={[{"type":"number","description":""}]}
          arguments={[{"name":"a","type":"object","description":""},{"name":"b","type":"object","description":""}]}
        />
      </td>

        `sortingOrder` <h5>número</h5>

      <td>
        Establece el orden de clasificación. Un número más bajo indica una prioridad de clasificación más alta, lo que significa que `0` será la primera columna ordenada y, si dos elementos son idénticos, usaremos la columna `1` y así sucesivamente. Las columnas que no tienen un orden de clasificación explícito se emplean luego de todas las columnas que sí lo tienen y se aplican en orden de aparición.

      <td>
        `value` <h5>función</h5>
      </td>

      <td>
        Función que se emplea para extraer el valor bruto que representa una columna. El valor sin procesar puede ser un booleano (por ejemplo, si representa una casilla de verificación), un número (si representa una métrica) o texto para otros. Se llama con un objeto que contiene el elemento que representa la fila, su índice en `items` matriz pasada a la tabla y los elementos mismos. Esta función se emplea para ordenar la tabla (por el valor dado), así como para la heurística de anchos `'fit-content'` .
      </td>
    </tr>

    <tr>
      <td>
        `width` <h5>cadena</h5>
      </td>

      <td>
        Representa el ancho de la columna. Siempre es una cadena y puede ser de uno de los siguientes tipos:

        * `'fit-content'`: la columna ocupará el espacio mínimo requerido para asignar todos los elementos de la columna. Para listas virtualizadas, se empleará una heurística para evitar representar todas las celdas.
        * `'YYpx'`: donde `YY` es un número, representa el ancho de la columna en pixeles.
        * `'YY%'`: donde `YY` es un número, representa el ancho de la columna en porcentaje, en relación con el tamaño del espacio disponible donde se representa la tabla.
        * `'YYfr'`: donde `YY` es un número, representa una fracción del ancho disponible. Las fracciones son proporcionales a otras; Por ejemplo, una tabla con dos columnas (`1fr` y `2fr`) tendrá la segunda columna dos veces más grande que la primera. Si, luego de calcular los anchos, el tamaño total de la tabla no cubre el espacio disponible, Todas las columnas se estirarán proporcionalmente para adaptar a todo el espacio disponible.
      </td>
    </tr>
