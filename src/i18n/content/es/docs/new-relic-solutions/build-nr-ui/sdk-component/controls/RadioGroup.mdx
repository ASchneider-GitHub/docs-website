Un grupo de `<Radio>` botones. Los botones `<Radio>` pueden ser hijos directos o descendientes del grupo de radio. Los botones `<Radio>` dentro de un grupo de radio deben tener un `value` único asignado.

Una vez que se establece un grupo de radio, al seleccionar cualquier `<Radio>` en ese grupo, se anula automáticamente la selección de cualquier `<Radio>` actualmente seleccionado en el grupo.

**Nota**: La configuración `value` anulará `defaultValue` ya que coloca `RadioGroup` en un estado controlado. `value` no anulará los componentes `Radio` que tengan `checked` configurado, ya que `RadioGroup` tomará el control de los componentes `Radio` no controlados y no puede anular los controlados.

El controlador de eventos `onChange` para `RadioGroup` se activará luego de cualquier controlador de eventos `onChange` configurado en componentes `Radio` individuales. Sin embargo, se recomienda encarecidamente configurar solo un controlador de eventos para todo el grupo en un `RadioGroup` controlado.

### Uso

```js
import { RadioGroup } from 'nr1'
```

### Ejemplos

#### Básico

```js
<RadioGroup defaultValue="2">
  <Radio label="Radio 1" value="1" />
  <Radio label="Radio 2" value="2" />
  <Radio label="Radio 3" value="3" />
</RadioGroup>
```

#### Con etiqueta e información.

```js
<RadioGroup label="Radio Group" info="Info value" defaultValue="2">
  <Radio label="Radio 1" value="1" />
  <Radio label="Radio 2" value="2" />
  <Radio label="Radio 3" value="3" />
</RadioGroup>
```

#### Con etiqueta en línea

```js
<RadioGroup label="Radio Group" labelInline defaultValue="2">
  <Radio label="Radio 1" value="1" />
  <Radio label="Radio 2" value="2" />
  <Radio label="Radio 3" value="3" />
</RadioGroup>
```

#### Con descripción

```js
<RadioGroup
  label="Radio Group"
  description="Description value"
  defaultValue="2"
>
  <Radio label="Radio 1" value="1" />
  <Radio label="Radio 2" value="2" />
  <Radio label="Radio 3" value="3" />
</RadioGroup>
```

#### Con mensaje inválido

```js
<RadioGroup
  label="Radio Group"
  invalid="Invalid message value"
  defaultValue="2"
>
  <Radio label="Radio 1" value="1" />
  <Radio label="Radio 2" value="2" />
  <Radio label="Radio 3" value="3" />
</RadioGroup>
```

#### Componente controlado

```js
class MyNerdlet extends React.PureComponent {
  constructor(props) {
    super(props);


    this.state = {
      selectedValue: '2',
    };


    this.onChange = this.onChange.bind(this);
  }


  onChange(event, value) {
    this.setState((state) => {
      return { selectedValue: value };
    });
  }


  render() {
    return (
      <RadioGroup value={this.state.selectedValue} onChange={this.onChange}>
        <Radio label="Radio 1" value="1" />
        <Radio label="Radio 2" value="2" />
        <Radio label="Radio 3" value="3" />
      </RadioGroup>
    );
  }
}
```
