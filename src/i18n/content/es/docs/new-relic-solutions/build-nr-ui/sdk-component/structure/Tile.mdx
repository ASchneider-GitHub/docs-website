### Uso

```js
import { Tile } from 'nr1'
```

### Ejemplos

#### Azulejo básico

```js
<Tile onClick={console.log}>
  <HeadingText type={HeadingText.TYPE.HEADING_6}>
    Single location failure
  </HeadingText>
  <BlockText>
    Check your company’s style guides. Great copy that doesn’t fit with the rest
    of the interface or marketing materials can be jarring to users.
  </BlockText>
</Tile>
```

#### Azulejo deshabilitado

```js
<Tile onClick={console.log} disabled>
  <HeadingText type={HeadingText.TYPE.HEADING_6}>
    Single location failure
  </HeadingText>
  <BlockText>
    Check your company’s style guides. Great copy that doesn’t fit with the rest
    of the interface or marketing materials can be jarring to users.
  </BlockText>
</Tile>
```

#### Mosaico de enlace

```js
<Tile to="https://login.newrelic.com/login">Login</Tile>
```

Para omitir un valor, use `SPACING_TYPE.OMIT`.<OptionReference array>Tile.SPACING_TYPE.EXTRA_LARGETile.SPACING_TYPE.LARGETile.SPACING_TYPE.MEDIUMTile.SPACING_TYPE.NONEtile.SPACING_TYPE.OMITTile.SPACING_TYPE.SMALL</OptionReference>
