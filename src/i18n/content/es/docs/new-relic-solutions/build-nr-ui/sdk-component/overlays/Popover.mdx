`<Popover>` son superposiciones alternables para mostrar superposiciones pequeñas a pedido.

El componente `<Popover>` requiere que proporcione un activador con `<PopoverTrigger>` que manejará la apertura/cierre de la superposición, así como el contenido de la superposición hasta `<PopoverBody />`.

### Uso

```js
import { Popover } from 'nr1'
```

### Ejemplos

#### Básico

```js
<Popover>
  <PopoverTrigger>Trigger</PopoverTrigger>
  <PopoverBody>
    <BlockText>Body</BlockText>
  </PopoverBody>
</Popover>
```

#### Botón como disparador

```js
<Popover>
  <PopoverTrigger>
    <Button>Trigger</Button>
  </PopoverTrigger>
  <PopoverBody>
    <BlockText>Body</BlockText>
  </PopoverBody>
</Popover>
```

#### TextField como disparador

```js
<Popover>
  <PopoverTrigger>
    <TextField />
  </PopoverTrigger>
  <PopoverBody>
    <BlockText>Body</BlockText>
  </PopoverBody>
</Popover>
```

#### Con popoverpie de página

```js
<Popover>
  <PopoverTrigger>
    <Button>Trigger</Button>
  </PopoverTrigger>
  <PopoverBody>
    <Card style={{ width: '250px' }}>
      <CardBody>
        <HeadingText>Heading</HeadingText>
        <BlockText
          spacingType={[
            BlockText.SPACING_TYPE.MEDIUM,
            BlockText.SPACING_TYPE.NONE,
          ]}
        >
          Item content
        </BlockText>
        <BlockText>Item content</BlockText>
      </CardBody>
    </Card>
    <PopoverFooter style={{ textAlign: 'right' }}>Info message</PopoverFooter>
  </PopoverBody>
</Popover>
```

#### Popover controlado

```js
class ControlledPopover extends React.Component {
  constructor() {
    super(...arguments);


    this.state = {
      opened: false,
    };


    this._onChange = this._onChange.bind(this);
  }


  _onChange(evt, opened) {
    this.setState({ opened }, () =>
      console.log('controlled state', evt, this.state),
    );
  }


  render() {
    const { opened } = this.state;


    return (
      <Popover opened={opened} onChange={this._onChange}>
        <PopoverTrigger>
          <Button>Trigger</Button>
        </PopoverTrigger>
        <PopoverBody>
          <Card style={{ width: '250px' }}>
            <CardBody>
              <HeadingText>Heading</HeadingText>
              <BlockText
                spacingType={[
                  BlockText.SPACING_TYPE.MEDIUM,
                  BlockText.SPACING_TYPE.NONE,
                ]}
              >
                Item content
              </BlockText>
              <BlockText>Item content</BlockText>
            </CardBody>
          </Card>
          <PopoverFooter style={{ textAlign: 'right' }}>
            Info message
          </PopoverFooter>
        </PopoverBody>
      </Popover>
    );
  }
}
```
