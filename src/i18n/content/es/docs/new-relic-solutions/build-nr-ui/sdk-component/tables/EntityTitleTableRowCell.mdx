Representa una celda de tabla de filas que muestra el título de una entidad en individua. Esta celda toma el objeto de entidad como hijo, que a su vez es compatible con el objeto de entidad NerdGraph (necesita, como mínimo, los campos `name` y `reporting` ). Si se puede alertar, también debe consultar por `alertSeverity`.

### Uso

```js
import { EntityTitleTableRowCell } from 'nr1'
```

### Ejemplos

#### Básico

```js
function render() {
  const items = [
    {
      name: 'Login service',
      guid: 'MTIzNDU2fEZPT3xCQVJ8OTg3NjU0MzIx',
      alertSeverity: 'CRITICAL',
      reporting: true,
    },
  ];


  return (
    <Table items={items}>
      <TableHeader>
        <TableHeaderCell>Entity</TableHeaderCell>
      </TableHeader>


      {({ item }) => (
        <TableRow>
          <EntityTitleTableRowCell value={item} />
        </TableRow>
      )}
    </Table>
  );
}
```

#### Con valor adicional

```js
function render() {
  const items = [
    {
      name: 'Login service',
      guid: 'MTIzNDU2fEZPT3xCQVJ8OTg3NjU0MzIx',
      alertSeverity: 'CRITICAL',
      reporting: true,
    },
  ];


  return (
    <Table items={items} multivalue>
      <TableHeader>
        <TableHeaderCell>Entity</TableHeaderCell>
      </TableHeader>


      {({ item }) => (
        <TableRow>
          <EntityTitleTableRowCell value={item} additionalValue={item.guid} />
        </TableRow>
      )}
    </Table>
  );
}
```
