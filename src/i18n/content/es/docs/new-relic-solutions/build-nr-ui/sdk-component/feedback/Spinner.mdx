Los controles giratorios se emplean para indicarle al usuario que el contenido se está cargando en la pantalla.

### Uso

```js
import { Spinner } from 'nr1'
```

### Ejemplos

#### Básico

```js
<Spinner />
```

#### Punto

```js
<Spinner type={Spinner.TYPE.DOT} />
```

#### En línea

```js
<Spinner inline />
```
