[Reacciona el contexto](https://reactjs.org/docs/context.html#when-to-use-context) para acceder al estado de la URL del nerdlet.

### Uso

```js
import { NerdletStateContext } from 'nr1'
```

### Ejemplos

#### Básico

```js
<NerdletStateContext.Consumer>
  {(nerdletState) => <pre>{JSON.stringify(nerdletState, null, 4)}</pre>}
</NerdletStateContext.Consumer>
```
