Elemento secundario del componente `<Popover>` .

Contiene el contenido de la superposición emergente.

### Uso

```js
import { PopoverBody } from 'nr1'
```

### Props

        `children` <h5>Nodo</h5> <h5>REQUERIDO</h5>
      </td>

      <td>
        Contenido que se mostrará cuando se abra Popover.
      </td>
    </tr>

      </td>

      <td>
        `onClose` <h5>función</h5>
      </td>

      <td>

        <FunctionDefinition
          returnValue={[]}
          arguments={[{"name":"event","type":"React.MouseEvent","description":""}]}
        />

        `onOpen` <h5>función</h5>
      </td>

      <td>
        La devolución de llamada se activa cada vez que se abre la superposición cuando `Popover` no está controlado. No se activa cuando se controla `Popover` .

        <FunctionDefinition
          returnValue={[]}
          arguments={[{"name":"event","type":"React.MouseEvent","description":""}]}
        />
      </td>
    </tr>

    <tr>
      <td>
        `onToggle` <h5>función</h5>
      </td>

      <td>
        La devolución de llamada se activa cada vez que se alterna la superposición cuando `Popover` no está controlado. No se activa cuando se controla `Popover` .

        <FunctionDefinition
          returnValue={[]}
          arguments={[{"name":"event","type":"React.MouseEvent","description":""},{"name":"opened","type":"boolean","description":""}]}
        />
      </td>
    </tr>

    <tr>
      <td>
        `placementType` <h5>enumeración</h5>
      </td>

      <td>
        Especifica la ubicación del cuerpo del popover en relación con el activador del popover. La visualización predeterminada está en el inicio inferior, pero también puede optar por mostrarla desde el extremo inferior.

        <OptionReference>
          PopoverBody.PLACEMENT_TYPE.BOTTOM_END, PopoverBody.PLACEMENT_TYPE.BOTTOM_START
        </OptionReference>
      </td>
