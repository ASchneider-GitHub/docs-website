La información sobre herramientas se emplea para exponer información adicional que no está disponible en la UI. Puede usarlos para explicar qué es un elemento UI , explicar el propósito de una acción o proporcionar una definición de una palabra o frase.

### Uso

```js
import { Tooltip } from 'nr1'
```

### Ejemplos

#### Básico

```js
<Tooltip text="Search across all your accounts">
  <Button>Search</Button>
</Tooltip>
```

#### Con tipo de ubicación

```js
<Tooltip
  text="Explore your metrics, events and logs"
  placementType={Tooltip.PLACEMENT_TYPE.BOTTOM}
>
  <Button>Query your data</Button>
</Tooltip>
```

#### Con enlace de información adicional

```js
function render() {
  const additionalInfoLink = {
    label: 'See the docs',
    to: 'https://docs.newrelic.com',
  };


  return (
    <p>
      Welcome to
      <Tooltip
        text="Go to docs for more information"
        placementType={Tooltip.PLACEMENT_TYPE.BOTTOM}
        additionalInfoLink={additionalInfoLink}
      >
        New Relic One
      </Tooltip>
    </p>
  );
}
```
