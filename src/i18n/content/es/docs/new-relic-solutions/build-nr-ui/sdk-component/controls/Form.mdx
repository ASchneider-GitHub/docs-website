`<Form>` es un contenedor que muestra un conjunto de campos de entrada de usuario relacionados de forma estructurada. Puede contener uno o más de los siguientes componentes del formulario:

* `<TextField>`
* `<MultilineTextField>`
* `<Checkbox>`
* `<Radio>`
* `<RadioGroup>`
* `<Switch>`
* `<Select>`
* `<Dropdown>`

El componente admite diferentes tipos de diseños que se pueden especificar mediante la propiedad `layoutType` .

### Uso

```js
import { Form } from 'nr1'
```

### Ejemplos

#### Diseño predeterminado

```js
<Form>
  <TextField placeholder="AAA Production" label="Name" />
  <Select label="Select account">
    <SelectItem>Option 1</SelectItem>
    <SelectItem>Option 2</SelectItem>
  </Select>
  <TextField
    defaultValue="help-production-db"
    label="Help channel"
    invalid="Please enter a valid slack channel"
  />
  <MultilineTextField
    label="Description"
    info="Info tooltip"
    placeholder="Placeholder text"
  />
  <Button
    type={Button.TYPE.PRIMARY}
    sizeType={Button.SIZE_TYPE.SMALL}
    spacingType={[
      HeadingText.SPACING_TYPE.EXTRA_LARGE,
      HeadingText.SPACING_TYPE.NONE,
    ]}
  >
    Create dashboard
  </Button>
</Form>
```

#### Diseño dividido

```js
<Form layoutType={Form.LAYOUT_TYPE.SPLIT}>
  <TextField placeholder="AAA Production" label="Name" />
  <Select label="Select account">
    <SelectItem>Option 1</SelectItem>
    <SelectItem>Option 2</SelectItem>
  </Select>
  <TextField
    defaultValue="help-production-db"
    label="Help channel"
    invalid="Please enter a valid slack channel"
  />
  <MultilineTextField
    label="Description"
    info="Info tooltip"
    placeholder="Placeholder text"
  />
  <Button
    type={Button.TYPE.PRIMARY}
    sizeType={Button.SIZE_TYPE.SMALL}
    spacingType={[
      HeadingText.SPACING_TYPE.EXTRA_LARGE,
      HeadingText.SPACING_TYPE.NONE,
    ]}
  >
    Create dashboard
  </Button>
</Form>
```

#### Diseño dividido con tipo de tamaño

```js
<Form
  layoutType={Form.LAYOUT_TYPE.SPLIT}

  <TextField placeholder="AAA Production" label="Name" />
```
