Cuando un usuario puede elegir solo una de las opciones proporcionadas

El estado del botón de opción se puede controlar a través de la propiedad `checked` . Si el accesorio `checked` no está configurado, el botón de opción gestionará su propio estado.

Puede escuchar el evento del botón de opción a través de los accesorios `onClick` o `onChange` .

### Uso

```js
import { Radio } from 'nr1'
```

### Ejemplos

#### Básico

```js
<div className="nr1-Docs-prettify">
  <Radio onChange={(event) => alert('Foo')} label="Foo" />
  <Radio checked label="Bar" />
  <Radio checked disabled label="Baz" />
</div>
```

#### con informacion

```js
<Radio onChange={(event) => alert('Foo')} info="Info value" label="Foo" />
```

#### Con descripción

```js
<Radio
  onChange={(event) => alert('Foo')}
  description="Description value"
  label="Foo"
/>
```

#### Con mensaje inválido

```js
<Radio
  onChange={(event) => alert('Foo')}
  invalid="Invalid message value"
  label="Foo"
/>
```

#### Componente controlado

```js
class MyNerdlet extends React.PureComponent {
  constructor(props) {
    super(props);


    this.state = {
      isChecked: false,
    };


    this.onClick = this.onClick.bind(this);
  }


  onClick(event) {
    this.setState((state) => {
      return { isChecked: !state.isChecked };
    });
  }


  render() {
    return (
      <Radio
        checked={this.state.isChecked}
        onClick={this.onClick}
        label="Radio button"
      />
    );
  }
}
```
