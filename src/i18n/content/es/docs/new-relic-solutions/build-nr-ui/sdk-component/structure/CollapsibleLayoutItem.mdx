Elemento secundario del componente `<Layout>` usado solo para áreas del lado izquierdo o derecho, con la capacidad de contraerlas. Puede usar como un componente "controlado" (es decir, pasando el atributo booleano `collapsed` ) o, de forma predeterminada, gestionará su propio estado (no) colapsado. Se necesita un accesorio `triggerType`, que le permite elegir los activadores incorporados (el área lateral incluirá un botón que permite al usuario contraer el área y luego expandirla nuevamente) o personalizado, lo que significa que está usando su propio elemento UI en otro lugar para controlar el estado (no) colapsado.

Consulte [los `<Layout>` documentos](/docs/new-relic-solutions/build-nr-ui/sdk-component/structure/Layout/#examples) para ver ejemplos de uso.

El componente `<Layout>` solo admite un hijo de cada `type`, compartido entre `<CollapsibleLayoutItem>` y `<LayoutItem>`.

### Uso

```js
import { CollapsibleLayoutItem } from 'nr1'
```
