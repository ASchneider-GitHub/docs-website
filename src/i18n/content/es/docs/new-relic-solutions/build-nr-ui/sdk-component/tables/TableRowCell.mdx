Envuelve el contenido de una celda de la tabla ubicada en el cuerpo de la tabla. Se espera que este componente solo se emplee como hijo de `<TableRow>`. Las celdas se encargan automáticamente de elipsificar el contenido que no encaja y centrarlo verticalmente.

Este componente es la celda base. Para casos comunes, el SDK también proporciona un conjunto de celdas adicionales para que siempre se pinten de manera consistente en toda la UI (por ejemplo, nombre de entidad con un estado o un usuario).

### Uso

```js
import { TableRowCell } from 'nr1'
```

### Ejemplos

#### Celda de fila con enlaces externos e información adicional

```js
function render() {
  const items = [
    {
      dashboardName: 'Staging service - Errors',
      createdBy: 'john@doe.com',
      url: '//fake-url.com/dashboard1',
    },
    {
      dashboardName: 'Package Registry - Performance',
      createdBy: 'jane@doe.com',
      url: '//fake-url.com/dashboard2',
    },
  ];
  return (
    <Table items={items} multivalue>
      <TableHeader>
        <TableHeaderCell>Name</TableHeaderCell>
        <TableHeaderCell>Url</TableHeaderCell>
      </TableHeader>
      {({ item }) => (
        <TableRow>
          <TableRowCell additionalValue={`Created by ${item.createdBy}`}>
            {item.dashboardName}
          </TableRowCell>
          <TableRowCell>
            <Link to={item.url}>Go to the dashboard</Link>
          </TableRowCell>
        </TableRow>
      )}
    </Table>
  );
}
```
