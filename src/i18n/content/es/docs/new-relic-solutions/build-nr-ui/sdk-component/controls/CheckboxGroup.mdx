Un grupo de `<Checkbox>` botones. Los botones `<Checkbox>` pueden ser hijos directos o descendientes del grupo de casillas de verificación.

**Nota**: La configuración `value` anulará `defaultValue` ya que coloca el grupo de casillas de verificación en un estado controlado. `value` no anulará las casillas de verificación que tengan `checked` configuradas, ya que `CheckboxGroup` tomará el control de los componentes `Checkbox` no controlados y no puede anular los controlados. Sin embargo, `value` anulará `defaultChecked` en `Checkbox`.

El controlador de eventos `onChange` para `CheckboxGroup` se activará luego de cualquier controlador de eventos `onChange` configurado en componentes `Checkbox` individuales. Sin embargo, se recomienda encarecidamente configurar solo un controlador de eventos para todo el grupo en un `CheckboxGroup` controlado.

`Checkbox` Los componentes **deben** tener un `value` configurado para poder ser controlados por `CheckboxGroup`.

### Uso

```js
import { CheckboxGroup } from 'nr1'
```

### Ejemplos

#### Básico

```js
<CheckboxGroup label="My checkbox group">
  <Checkbox label="Checkbox 1" />
  <Checkbox label="Checkbox 2" />
  <Checkbox label="Checkbox 3" />
</CheckboxGroup>
```

#### Con etiqueta e información.

```js
<CheckboxGroup label="My checkbox group" info="Info value">
  <Checkbox label="Checkbox 1" />
  <Checkbox label="Checkbox 2" />
  <Checkbox label="Checkbox 3" />
</CheckboxGroup>
```

#### Con etiqueta en línea

```js
<CheckboxGroup label="My checkbox group" labelInline>
  <Checkbox label="Checkbox 1" />
  <Checkbox label="Checkbox 2" />
  <Checkbox label="Checkbox 3" />
</CheckboxGroup>
```

#### Con descripción

```js
<CheckboxGroup label="My checkbox group" description="Description value">
  <Checkbox label="Checkbox 1" />
  <Checkbox label="Checkbox 2" />
  <Checkbox label="Checkbox 3" />
</CheckboxGroup>
```

#### Con mensaje inválido

```js
<CheckboxGroup label="My checkbox group" invalid="Invalid message value">
  <Checkbox label="Checkbox 1" />
  <Checkbox label="Checkbox 2" />
  <Checkbox label="Checkbox 3" />
</CheckboxGroup>
```

#### Con valores predeterminados

```js
<CheckboxGroup label="My checkbox group" defaultValue={['1', '2']}>
  <Checkbox label="Checkbox 1" value="1" />
  <Checkbox label="Checkbox 2" value="2" />
  <Checkbox label="Checkbox 3" value="3" />
</CheckboxGroup>
```

#### Componente controlado

```js
class MyNerdlet extends React.PureComponent {
  constructor(props) {
    super(props);


    this.state = {
      values: ['2'],
    };


    this.onChange = this.onChange.bind(this);
  }


  onChange(event, values) {
    this.setState({ values });
  }


  render() {
    return (
      <CheckboxGroup value={this.state.values} onChange={this.onChange}>
        <Checkbox label="Checkbox 1" value="1" />
        <Checkbox label="Checkbox 2" value="2" />
        <Checkbox label="Checkbox 3" value="3" />
      </CheckboxGroup>
    );
  }
}
```
