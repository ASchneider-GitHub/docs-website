Proporciona espacio a cualquier componente envuelto. El espaciado se proporciona como una tupla que puede contener cero, uno, dos, tres o cuatro valores. Cada uno de estos valores proviene de la propiedad estática TYPE.

La estructura de la tupla es muy similar a cómo funcionan las propiedades `margin` o `padding` : al especificar un valor se replican en los cuatro lados, dos en `top`/`bottom` y `left`/`right`, etc. Si está dispuesto a omitir un valor específico y pasar otros, puede emplear `OMIT` como marcador de posición. Se dejará intacto el espacio del lado (o lados) donde se emplee.

Otros componentes del SDK tienen capacidades de espaciado integradas. Si desea replicar la misma capacidad en su propio componente, puede usar `Spacing.add` y pasar el componente; el componente devuelto por el método contendrá la propiedad `type` así como la estática.

### Uso

```js
import { Spacing } from 'nr1'
```

      <td>
        Propiedad de espaciado. El espaciado se define como una tupla de cero a cuatro valores, que siguen las mismas convenciones que las propiedades CSS como `margin` o `padding`. Para omitir un valor, use `TYPE.OMIT`.

        <OptionReference array>
          Espaciado.TIPO.EXTRA_LARGE, Espaciado.TIPO.LARGE, Espaciado.TIPO.MEDIO, Espaciado.TIPO.NINGUNO, Espaciado.TIPO.OMITIR, Espaciado.TIPO.PEQUEÑO,
        </OptionReference>

        Básico

        ```js
        <Spacing type={[Spacing.TYPE.MEDIUM]}>
          <Button>Spaced button</Button>
        </Spacing>;            
        ```

        Omitir márgenes

        ```js
        <Spacing type={[Spacing.TYPE.OMIT, Spacing.TYPE.LARGE]}>
          <Button>Spaced button (only left and right)</Button>
        </Spacing>; 
        ```
      </td>
