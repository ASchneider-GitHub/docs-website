[Contexto de reacción](https://reactjs.org/docs/context.html#when-to-use-context) para acceder al estado de la URL de la plataforma.

### Uso

```js
import { PlatformStateContext } from 'nr1'
```

### Ejemplos

#### Básico

```js
<PlatformStateContext.Consumer>
  {(platformState) => <pre>{JSON.stringify(platformState, null, 4)}</pre>}
</PlatformStateContext.Consumer>
```
