Envoltorio para celdas que componen una sola fila.

### Uso

```js
import { TableRow } from 'nr1'
```

### Props

<table>
  <tbody>
    <tr>
      <td>
        `actions` <h5>forma\[]</h5>
      </td>

      <td>
        Define un conjunto de acciones que aparecerán cuando el usuario pase el cursor sobre la fila. Las acciones tienen un texto obligatorio y onClick devolución de llamada (que se llamará con el ítem e índice de la fila a la que pertenecen), y también aceptan una propiedad deshabilitada.

        <h3>forma</h3>

        `disabled` <h5>booleano</h5>

        `label` <h5>Cadena</h5><h5>REQUIRED</h5> Úsela para describir la acción de la fila.

        `onClick` <h5>Función</h5><h5>REQUERIDA</h5> La devolución de llamada se activa cada vez que el usuario hace clic en la acción.

        <FunctionDefinition
          returnValue={[]}
          arguments={[{"name":"event","type":"React.MouseEvent","description":""}]}
        />

        `testId` <h5>cadena</h5> Agrega un atributo `data-test-id` . Úselo para apuntar al componente en pruebas unitarias y E2E. Para que una identificación de prueba sea válida, antepóngale su identificación de nerdpack, seguido de un punto. Por ejemplo, `my-nerdpack.some-element`. **Nota:** Es posible que no vea el atributo `data-test-id` ya que se eliminan del DOM. Para depurarlos, pase un parámetro de consulta `e2e-test` a la URL.

        `type` <h5>enum</h5> Tipo de acción para cada uno de los elementos en el menú desplegable de acciones. No es obligatorio, su valor predeterminado es `TableRow.ACTION_TYPE.NORMAL`.

        * Normal: acción genérica no crítica.

        * Destructivo: empléelo cuando tenga una acción destructiva como eliminar o eliminar, que le gustaría que el usuario pausara y considerara antes de completarla.

          <OptionReference>
            TableRow.ACTION_TYPE.DESTRUCTIVE, TableRow.ACTION_TYPE.NORMAL,
          </OptionReference>

        ```js
        const actions = [
          {
            label: 'Duplicate',
            type: TableRow.ACTION_TYPE.NORMAL,
          },
          {
            label: 'Delete',
            type: TableRow.ACTION_TYPE.DESTRUCTIVE,
          },
        ];
        ```

        ```js
        const actions = [
          {
            label: 'Action 1',
            disabled: true,
            onClick: (evt, { item, index }) => {
              alert(`Action 1:Item: ${index}${JSON.stringify(item, null, 2)}`);
            },
          },
          {
            label: 'Action 2',
            onClick: (evt, { item, index }) => {
              alert(`Action 2:Item: ${index}${JSON.stringify(item, null, 2)}`);
            },
          },
        ];
        ```
      </td>

      </td>

        />
