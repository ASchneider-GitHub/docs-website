Representa una celda de una tabla de filas que muestra una métrica particular. La métrica tiene el formato según la unidad pasada. Las unidades son compatibles 1:1 con las compatibles con el tipo de formato `<NrqlQuery>` `CHART`.

El formato de unidad y prefijo se selecciona automáticamente para que sea coherente con el resto de la plataforma. métricas se alinean automáticamente hacia la izquierda y se elipsan desde la derecha, que es la forma estándar de presentarlas.

### Uso

```js
import { MetricTableRowCell } from 'nr1'
```

### Ejemplos

#### Básico

```js
function render() {
  const items = [
    {
      value: 123456789,
    },
  ];


  return (
    <Table items={items}>
      <TableHeader>
        <TableHeaderCell alignmentType={TableHeaderCell.ALIGNMENT_TYPE.RIGHT}>
          Memory usage
        </TableHeaderCell>
      </TableHeader>


      {({ item }) => (
        <TableRow>
          <MetricTableRowCell
            type={MetricTableRowCell.TYPE.BITS}
            value={item.value}
          />
        </TableRow>
      )}
    </Table>
  );
}
```

#### Con valor adicional

```js
function render() {
  const items = [
    {
      value: 123456789,
      lastUpdated: new Date().toLocaleTimeString(),
    },
  ];


  return (
    <Table items={items} multivalue>
      <TableHeader>
        <TableHeaderCell alignmentType={TableHeaderCell.ALIGNMENT_TYPE.RIGHT}>
          Memory usage
        </TableHeaderCell>
      </TableHeader>


      {({ item }) => (
        <TableRow>
          <MetricTableRowCell
            type={MetricTableRowCell.TYPE.BITS}
            value={item.value}
            additionalValue={`Last updated at ${item.lastUpdated}`}
          />
        </TableRow>
      )}
    </Table>
  );
}
```
