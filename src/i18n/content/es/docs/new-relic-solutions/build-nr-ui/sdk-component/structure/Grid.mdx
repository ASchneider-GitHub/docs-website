Grid es un sistema de diseño bidimensional, lo que significa que puede manejar tanto columnas como filas, lo que lo ayuda a diseñar una interfaz de usuario basada en cuadrículas, a diferencia de `<Stack>` , que es en gran medida un sistema unidimensional.

### Uso

```js
import { Grid } from 'nr1'
```
