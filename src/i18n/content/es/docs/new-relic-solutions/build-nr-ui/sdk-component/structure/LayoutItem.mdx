Elemento secundario del componente `<Layout>` .

De forma predeterminada, emplea el tipo principal para abarcar todo el ancho del área de Diseño. Dividir a la izquierda y dividir a la derecha proporcionan un área fija izquierda/derecha. El diseño solo admite un tipo de cada uno.

### Uso

```js
import { LayoutItem } from 'nr1'
```
