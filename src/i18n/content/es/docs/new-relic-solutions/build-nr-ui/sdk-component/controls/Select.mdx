El componente `<Select>` es una variante del `<Dropdown>` , especialmente destinado a ser empleado en componentes de formulario. Las principales diferencias son:

* Sólo puede funcionar en modo declarativo, es decir, no se puede virtualizar pasando una función secundaria.
* Cada `<SelectItem>` acepta una propiedad `value` , que se comparará con el `value` pasado a `<Select>` y se seleccionará la opción correcta. A diferencia de un elemento DOM `<SELECT>` tradicional, cualquier referencia se puede emplear como valor, tanto primitivos como objetos.
* Los artículos no aceptan un `onClick`. En su lugar, obtendrá el valor de la opción seleccionada al recibirlo de un método `onChange` .

El componente siempre está controlado, lo que significa que debe actualizar su `value` desde una devolución de llamada `onChange` para que refleje la propiedad recién seleccionada.

### Uso

```js
import { Select } from 'nr1'
```

### Ejemplos

#### Básico

```js
<Select onChange={(evt, value) => alert(value)}>
  <SelectItem value="a">Value is "a"</SelectItem>
  <SelectItem value="b">Value is "b"</SelectItem>
  <SelectItem value="c">Value is "c"</SelectItem>
</Select>
```

#### Con etiqueta e información.

```js
<Select label="Items" info="Info value" onChange={(evt, value) => alert(value)}>
  <SelectItem value="a">Value is "a"</SelectItem>
  <SelectItem value="b">Value is "b"</SelectItem>
  <SelectItem value="c">Value is "c"</SelectItem>
</Select>
```

#### Con etiqueta en línea

```js
<Select label="Items" labelInline onChange={(evt, value) => alert(value)}>
  <SelectItem value="a">Value is "a"</SelectItem>
  <SelectItem value="b">Value is "b"</SelectItem>
  <SelectItem value="c">Value is "c"</SelectItem>
</Select>
```

#### Con descripción

```js
<Select description="Description value" onChange={(evt, value) => alert(value)}>
  <SelectItem value="a">Value is "a"</SelectItem>
  <SelectItem value="b">Value is "b"</SelectItem>
  <SelectItem value="c">Value is "c"</SelectItem>
</Select>
```

#### Con mensaje inválido

```js
<Select invalid="Invalid message value" onChange={(evt, value) => alert(value)}>
  <SelectItem value="a">Value is "a"</SelectItem>
  <SelectItem value="b">Value is "b"</SelectItem>
  <SelectItem value="c">Value is "c"</SelectItem>
</Select>
```

#### Componente controlado

```js
class ControlledSelect extends React.Component {
  constructor() {
    super(...arguments);


    this.state = {
      value: null,
    };


    this._onChange = this._onChange.bind(this);
  }


  _onChange(event, value) {
    this.setState({ value });
  }


  render() {
    return (
      <Select onChange={this._onChange} value={this.state.value}>
        <SelectItem value="1">Item 1</SelectItem>
        <SelectItem value="2">Item 2</SelectItem>
        <SelectItem value="3">Item 3</SelectItem>
      </Select>
    );
  }
}
```
