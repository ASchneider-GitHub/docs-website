Recupera todas las cuentas y permite la selección de cuentas mediante menú desplegable y búsqueda. Para capturar la cuenta seleccionada actualmente, puede emplear el oyente `onChange` .

Tenga en cuenta que la plataforma ya contiene su propio selector de cuentas, que se encuentra en el encabezado de la misma. Este se puede configurar para que se muestre usando la API `nerdlet.setConfig` (pasando `{ accountPicker: true }`) y su valor se puede leer usando `<PlatformStateContext.Consumer>`. Para obtener una referencia completa, consulte estas API.

### Uso

```js
import { AccountPicker } from 'nr1'
```

### Ejemplos

#### Básico

```js
class Example extends React.Component {
  constructor() {
    super(...arguments);


    this.state = { accountId: null };


    this.onChangeAccount = this.onChangeAccount.bind(this);
  }


  onChangeAccount(_, value) {
    alert(`Selected account: ${value}`);


    this.setState({ accountId: value });
  }


  render() {
    return (
      <AccountPicker
        value={this.state.accountId}
        onChange={this.onChangeAccount}
      />
    );
  }
}
```

#### Con etiqueta en línea

```js
class Example extends React.Component {
  constructor() {
    super(...arguments);


    this.state = { accountId: null };


    this.onChangeAccount = this.onChangeAccount.bind(this);
  }


  onChangeAccount(_, value) {
    alert(`Selected account: ${value}`);


    this.setState({ accountId: value });
  }


  render() {
    return (
      <AccountPicker
        label="Account"
        labelInline
        value={this.state.accountId}
        onChange={this.onChangeAccount}
      />
    );
  }
}
```
