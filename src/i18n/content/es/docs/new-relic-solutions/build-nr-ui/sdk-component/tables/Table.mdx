Representa una tabla con un encabezado y filas fijos. La tabla implementa la siguiente característica:

* Diseño flexible: los encabezados de las tablas aceptan una variedad de tamaños para permitir diseños fluidos y fijos. Puedes encontrar más información sobre cómo personalizar tus columnas consultando `TableHeaderCell`.
* Clasificación: la tabla puede ordenar internamente los elementos pasados de acuerdo con el estado de clasificación actual de la tabla. Puede encontrar más información sobre la clasificación marcando `TableHeaderCell`.
* Selección de filas: las filas se pueden seleccionar mediante las casillas de verificación en el lado derecho. Puede encontrar más información en la propiedad `selected` .
* Acciones de fila: las acciones contextuales se pueden activar por fila, lo que permite al usuario lograr funcionalidad sobre ellas. Puede encontrar más información sobre cómo agregarlos en `TableRow`.
* Celdas predefinidas personalizadas: algunos patrones muy comunes para las celdas (título de entidad, métrica, etc.) ya están proporcionados por la plataforma, por lo que solo necesitas devolverlos como parte de tu fila.
* Virtualización: las celdas sólo se renderizan si se muestran en pantalla. Esto permite que la tabla funcione con un gran conjunto de datos casi sin penalización en el rendimiento.

### Uso

```js
import { Table } from 'nr1'
```

#### Clasificación de una sola columna

```js
class Example extends React.Component {
  constructor(...args) {
    super(...args);


    this.state = {
      column: 0,
      sortingType: TableHeaderCell.SORTING_TYPE.NONE,
    };


    this.exampleItems = [
      { x: 'A', y: 0 },
      { x: 'B', y: 0 },
      { x: 'A', y: 1 },
    ];
  }


  _onClickTableHeaderCell(column, evt, { nextSortingType }) {
    if (column === this.state.column) {
      this.setState({ sortingType: nextSortingType });
    } else {
      this.setState({ column: column, sortingType: nextSortingType });
    }
  }


  render() {
    const sortingType0 =
      this.state.column === 0
        ? this.state.sortingType
        : TableHeaderCell.SORTING_TYPE.NONE;


    const sortingType1 =
      this.state.column === 1
        ? this.state.sortingType
        : TableHeaderCell.SORTING_TYPE.NONE;


    return (
      <Table items={this.exampleItems}>
        <TableHeader>
          <TableHeaderCell
            sortable
            sortingType={sortingType0}
            onClick={this._onClickTableHeaderCell.bind(this, 0)}
            value={({ item }) => item.x}
          >
            Column 1
          </TableHeaderCell>


          <TableHeaderCell
            sortable
            sortingType={sortingType1}
            onClick={this._onClickTableHeaderCell.bind(this, 1)}
            value={({ item }) => item.y}
          >
            Column 2
          </TableHeaderCell>
        </TableHeader>


        {({ item }) => (
          <TableRow>
            <TableRowCell>{item.x}</TableRowCell>
            <TableRowCell>{item.y}</TableRowCell>
          </TableRow>
        )}
      </Table>
    );
  }
}
```
