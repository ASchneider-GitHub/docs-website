Elemento secundario del componente `<CardBody>` .

### Uso

```js
import { CardSection } from 'nr1'
```

### Props

<table>
  <tbody>
    <tr>
      <td>
        `children` <h5>Nodo</h5> <h5>REQUERIDO</h5>
      </td>

      <td>
        Contenido para renderizar dentro de la tarjeta.
      </td>
    </tr>

    <tr>
      <td>
        `className` <h5>cadena</h5>
      </td>

      <td>
        Agrega nombres de clase al componente. Debe usar solo con fines de posicionamiento y espaciado.
      </td>
    </tr>

    <tr>
      <td>
        `collapsed` <h5>booleano</h5>
      </td>

      <td>
        Configure este accesorio para controlar el estado colapsado del componente. Si se define, convierte el componente en un [componente controlado](https://facebook.github.io/react/docs/forms.html).
      </td>
    </tr>

    <tr>
      <td>
        `collapsible` <h5>booleano</h5>
      </td>

      <td>
        Configure este accesorio para que el componente sea plegable.
      </td>
    </tr>

    <tr>
      <td>
        `defaultCollapsed` <h5>booleano</h5>
      </td>

      <td>
        Si es `true`, el estado inicial contraído de la tarjeta es `collapsed`. Es útil cuando no desea emplear un [componente controlado](https://facebook.github.io/react/docs/forms.html).
      </td>
    </tr>

    <tr>
      <td>
        `onChange` <h5>función</h5>
      </td>

      <td>
        Esta función se llamará cada vez que el usuario haga clic en el encabezado para contraer o expandir el componente. Si está controlando el estado del componente, empléelo para establecer el valor de la propiedad `collapsed` .

        <FunctionDefinition
          returnValue={[]}
          arguments={[{"name":"event","type":"React.MouseEvent","description":""},{"name":"collapsed","type":"boolean","description":""}]}
        />
      </td>
    </tr>

    <tr>
      <td>
        `style` <h5>objeto</h5>
      </td>

      <td>
        Estilo en línea para un estilo personalizado. Debe usar solo con fines de posicionamiento y espaciado.
      </td>
    </tr>

    <tr>
      <td>
        `testId` <h5>cadena</h5>
      </td>

      <td>
        Agrega un atributo `data-test-id` . Úselo para apuntar al componente en pruebas unitarias y E2E. Para que una identificación de prueba sea válida, antepóngale su identificación de nerdpack, seguido de un punto. Por ejemplo, `my-nerdpack.some-element`.

        **Nota:** Es posible que no veas el atributo `data-test-id` ya que se eliminan del DOM. Para depurarlos, pasa un parámetro de consulta `e2e-test` a la URL.
      </td>
    </tr>
  </tbody>
</table>
