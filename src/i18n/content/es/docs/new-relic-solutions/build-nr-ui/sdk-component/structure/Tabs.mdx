Las pestañas se emplean para ayudar a agrupar diversos contenidos relacionados en secciones separadas. Esas secciones pueden ser todas o estar contenidas en una parte de la pantalla. Recomendamos usar un conjunto de pestañas por pantalla y también usarlo solo para diferentes tipos de contenido relacionado. Por ejemplo, cuando tiene datos diferentes sobre una sola entidad, puede agrupar los datos en diferentes secciones.

NO separe una acción, como crear, de otras secciones de contenido en pestañas. La acción debe activar mediante un icono o botón y abrir una página modal o apilada.

### Uso

```js
import { Tabs } from 'nr1'
```

### Ejemplos

#### Básico

```js
<Tabs defaultValue="tab-3">
  <TabsItem value="tab-1" label="Tab 1">
    Tab 1 content
  </TabsItem>
  <TabsItem value="tab-2" label="Another tab with a longer name">
    Tab 2 content
  </TabsItem>
  <TabsItem value="tab-3" label="A tab open by default">
    Tab 3 content
  </TabsItem>
</Tabs>
```

#### Renderizar devolución de llamada

```js
function render() {
  const items = new Array(10000).fill().map((_, i) => ({
    value: i,
    content: `Tab ${i} content...`,
    label: `Item ${i}`,
  }));


  return (
    <div className="nr1-Example--tabs">
      <Tabs items={items}>
        {({ item }) => (
          <TabsItem value={item.value} label={item.label}>
            {item.content}
          </TabsItem>
        )}
      </Tabs>
    </div>
  );
}
```
