Representa una celda de una tabla de filas que muestra un usuario en individuo. Esta celda toma el objeto de usuario como hijo, que a su vez es compatible con el objeto de usuario NerdGraph (necesita, como mínimo, los campos `name` y `gravatar` ).

### Uso

```js
import { UserTableRowCell } from 'nr1'
```

### Ejemplos

#### Ejemplo 1

```js
function render() {
  const items = [
    {
      name: 'John Doe',
      gravatar:
        'https://secure.gravatar.com/avatar/79988df068cb4d952284a3395412dc4f',
    },
  ];


  return (
    <Table items={items}>
      <TableHeader>
        <TableHeaderCell>User</TableHeaderCell>
      </TableHeader>


      {({ item }) => (
        <TableRow>
          <UserTableRowCell value={item} />
        </TableRow>
      )}
    </Table>
  );
}
```
