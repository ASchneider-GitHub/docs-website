<Callout variant="important" title="AVISO EOL">
  La API de asociación llegó al final de su vida útil (EOL) en marzo de 2023 y ya no es compatible ni está disponible para su uso.
</Callout>

Nuestra API de asociación permitió a los socios New Relic y a las cuentas New Relic configuradas como [asociaciones de clientes](/docs/new-relic-partnerships/partner-integration-guide/partner-account-maintenance/partnership-accounts-users-subscriptions/#partnerships) gestionar cuentas, usuarios y configuraciones relacionadas con la subscripción.

Dado que la API de asociación ya no está disponible, puede emplear nuestra API Nerdgraph para realizar muchas de las mismas tareas:

* [Cuentas de administración](/docs/apis/nerdgraph/examples/manage-accounts-nerdgraph)
* [Administrar usuario](/docs/apis/nerdgraph/examples/nerdgraph-manage-users)
* [Administrar grupos de usuarios](/docs/apis/nerdgraph/examples/nerdgraph-manage-groups)
