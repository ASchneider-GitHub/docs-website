Wrapper para células que compõem uma única linha.

### Uso

```js
import { TableRow } from 'nr1'
```

### Props

<table>
  <tbody>
    <tr>
      <td>
        `actions` <h5>forma\[]</h5>
      </td>

      <td>
        Define um conjunto de ações que aparecerão quando o usuário passar o mouse sobre a linha. As ações possuem texto obrigatório e retorno de chamada onClick (que será chamado com o item e índice da linha a que pertencem), e também aceitam uma propriedade desabilitada.

        <h3>forma</h3>

        `disabled` <h5>boleano</h5>

        `label` <h5>String</h5><h5>OBRIGATÓRIA</h5> Use-a para descrever a ação da linha.

        `onClick` <h5>Função</h5><h5>REQUIRED</h5> retorno de chamada disparada sempre que o usuário clica na ação.

        <FunctionDefinition
          returnValue={[]}
          arguments={[{"name":"event","type":"React.MouseEvent","description":""}]}
        />

        `testId` <h5>string</h5> Adiciona um atributo `data-test-id` . Use-o para destino do componente em testes unitários e E2E. Para que um ID de teste seja válido, prefixe-o com seu ID nerdpack, seguido por um ponto. Por exemplo, `my-nerdpack.some-element`. **Nota:** Você pode não ver `data-test-id` atributo pois eles são removidos do DOM, para depurá-los passe um parâmetro de consulta `e2e-test` para a URL.

        `type` <h5>enum</h5> Tipo de ação para cada um dos itens no dropdown de ações. Não é obrigatório, seu padrão é `TableRow.ACTION_TYPE.NORMAL`.

        * Normal — ação genérica não crítica.

        * Destrutivo – use quando você tiver uma ação destrutiva, como excluir ou remover, que gostaria que o usuário pausasse e considerasse antes de concluir.

          <OptionReference>
            TableRow.ACTION_TYPE.DESTRUCTIVE, TableRow.ACTION_TYPE.NORMAL,
          </OptionReference>

        ```js
        const actions = [
          {
            label: 'Duplicate',
            type: TableRow.ACTION_TYPE.NORMAL,
          },
          {
            label: 'Delete',
            type: TableRow.ACTION_TYPE.DESTRUCTIVE,
          },
        ];
        ```

        ```js
        const actions = [
          {
            label: 'Action 1',
            disabled: true,
            onClick: (evt, { item, index }) => {
              alert(`Action 1:Item: ${index}${JSON.stringify(item, null, 2)}`);
            },
          },
          {
            label: 'Action 2',
            onClick: (evt, { item, index }) => {
              alert(`Action 2:Item: ${index}${JSON.stringify(item, null, 2)}`);
            },
          },
        ];
        ```
      </td>

        Lista de células de linha, no formato de `<TableRowCell>`s. As células precisam ser filhas diretas da linha, ou seja, não deve haver nenhum nó entre `<TableRow>` e `<TableRowCell>`s.
      </td>

        <FunctionDefinition
          returnValue={[]}
          arguments={[{"name":"event","type":"React.MouseEvent","description":""}]}
        />
      </td>
