Elemento filho do componente `<Popover>` .

Contém o conteúdo da sobreposição Popover.

### Uso

```js
import { PopoverBody } from 'nr1'
```

        <FunctionDefinition
          returnValue={[]}

        />
      </td>

        <FunctionDefinition
          returnValue={[]}
