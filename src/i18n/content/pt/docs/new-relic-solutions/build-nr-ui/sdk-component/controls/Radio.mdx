Quando um usuário pode escolher apenas uma das opções fornecidas

O estado do botão de opção pode ser controlado por meio da propriedade `checked` . Se a propriedade `checked` não estiver definida, o botão de opção gerenciará seu próprio estado.

Você pode ouvir o evento do botão de opção por meio dos adereços `onClick` ou `onChange` .

### Uso

```js
import { Radio } from 'nr1'
```

### Exemplos

#### Básico

```js
<div className="nr1-Docs-prettify">
  <Radio onChange={(event) => alert('Foo')} label="Foo" />
  <Radio checked label="Bar" />
  <Radio checked disabled label="Baz" />
</div>
```

#### Com informações

```js
<Radio onChange={(event) => alert('Foo')} info="Info value" label="Foo" />
```

#### Com descrição

```js
<Radio
  onChange={(event) => alert('Foo')}
  description="Description value"
  label="Foo"
/>
```

#### Com mensagem inválida

```js
<Radio
  onChange={(event) => alert('Foo')}
  invalid="Invalid message value"
  label="Foo"
/>
```

#### Componente controlado

```js
class MyNerdlet extends React.PureComponent {
  constructor(props) {
    super(props);


    this.state = {
      isChecked: false,
    };


    this.onClick = this.onClick.bind(this);
  }


  onClick(event) {
    this.setState((state) => {
      return { isChecked: !state.isChecked };
    });
  }


  render() {
    return (
      <Radio
        checked={this.state.isChecked}
        onClick={this.onClick}
        label="Radio button"
      />
    );
  }
}
```
