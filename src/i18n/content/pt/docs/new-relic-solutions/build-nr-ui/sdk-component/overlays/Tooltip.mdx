As dicas de ferramentas são usadas para expor informações adicionais que não estão prontamente disponíveis na interface. Você pode usá-los para explicar o que é um elemento de interface, explicar o propósito de uma ação ou fornecer a definição de uma palavra ou frase.

### Uso

```js
import { Tooltip } from 'nr1'
```

### Exemplos

#### Básico

```js
<Tooltip text="Search across all your accounts">
  <Button>Search</Button>
</Tooltip>
```

#### Com posicionamentoType

```js
<Tooltip
  text="Explore your metrics, events and logs"
  placementType={Tooltip.PLACEMENT_TYPE.BOTTOM}
>
  <Button>Query your data</Button>
</Tooltip>
```

#### Com link de informações adicionais

```js
function render() {
  const additionalInfoLink = {
    label: 'See the docs',
    to: 'https://docs.newrelic.com',
  };


  return (
    <p>
      Welcome to
      <Tooltip
        text="Go to docs for more information"
        placementType={Tooltip.PLACEMENT_TYPE.BOTTOM}
        additionalInfoLink={additionalInfoLink}
      >
        New Relic One
      </Tooltip>
    </p>
  );
}
```
