As guias são usadas para ajudar a agrupar vários conteúdos relacionados em seções separadas. Essas seções podem ser todas ou estar contidas em uma parte da tela. Recomendamos usar um conjunto de guias por tela e também usá-lo apenas para diferentes tipos de conteúdo relacionado. Por exemplo, quando você tem dados diferentes sobre uma única entidade, você pode agrupar os dados em seções diferentes.

NÃO separe uma ação, como criar, de outras seções de conteúdo em guias. A ação deve ser acionada por um ícone ou botão e abrir uma página modal ou empilhada.

### Uso

```js
import { Tabs } from 'nr1'
```

### Exemplos

#### Básico

```js
<Tabs defaultValue="tab-3">
  <TabsItem value="tab-1" label="Tab 1">
    Tab 1 content
  </TabsItem>
  <TabsItem value="tab-2" label="Another tab with a longer name">
    Tab 2 content
  </TabsItem>
  <TabsItem value="tab-3" label="A tab open by default">
    Tab 3 content
  </TabsItem>
</Tabs>
```

#### Renderizar retorno de chamada

```js
function render() {
  const items = new Array(10000).fill().map((_, i) => ({
    value: i,
    content: `Tab ${i} content...`,
    label: `Item ${i}`,
  }));


  return (
    <div className="nr1-Example--tabs">
      <Tabs items={items}>
        {({ item }) => (
          <TabsItem value={item.value} label={item.label}>
            {item.content}
          </TabsItem>
        )}
      </Tabs>
    </div>
  );
}
```
