Renderiza uma célula da tabela de linhas mostrando uma métrica específica. A métrica é formatada de acordo com a unidade passada. As unidades são compatíveis 1:1 com aquelas suportadas pelo tipo de formato `<NrqlQuery>` `CHART`.

A formatação de unidade e prefixo é escolhida automaticamente para que seja consistente com o resto da plataforma. As métricas são automaticamente alinhadas à esquerda e reticuladas à direita, que é a forma padrão de apresentação.

### Uso

```js
import { MetricTableRowCell } from 'nr1'
```

### Exemplos

#### Básico

```js
function render() {
  const items = [
    {
      value: 123456789,
    },
  ];


  return (
    <Table items={items}>
      <TableHeader>
        <TableHeaderCell alignmentType={TableHeaderCell.ALIGNMENT_TYPE.RIGHT}>
          Memory usage
        </TableHeaderCell>
      </TableHeader>


      {({ item }) => (
        <TableRow>
          <MetricTableRowCell
            type={MetricTableRowCell.TYPE.BITS}
            value={item.value}
          />
        </TableRow>
      )}
    </Table>
  );
}
```

#### Com valor adicional

```js
function render() {
  const items = [
    {
      value: 123456789,
      lastUpdated: new Date().toLocaleTimeString(),
    },
  ];


  return (
    <Table items={items} multivalue>
      <TableHeader>
        <TableHeaderCell alignmentType={TableHeaderCell.ALIGNMENT_TYPE.RIGHT}>
          Memory usage
        </TableHeaderCell>
      </TableHeader>


      {({ item }) => (
        <TableRow>
          <MetricTableRowCell
            type={MetricTableRowCell.TYPE.BITS}
            value={item.value}
            additionalValue={`Last updated at ${item.lastUpdated}`}
          />
        </TableRow>
      )}
    </Table>
  );
}
```
