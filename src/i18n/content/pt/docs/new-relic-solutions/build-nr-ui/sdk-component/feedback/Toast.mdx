A notificação do brinde dá ao usuário a confirmação imediata da conclusão bem-sucedida de uma tarefa ou falha crítica na tentativa de conclusão de uma tarefa.

Você pode personalizar o brinde passando ações personalizadas que serão exibidas como botões clicáveis ao lado do texto.

### Uso

```js
import { Toast } from 'nr1'
```

### Exemplos

#### Notificação básica de brinde

```js
function render() {
  const onClick = () =>
    Toast.showToast({
      title: 'Feedback received',
      description: 'We will contact you soon.',
      actions: [
        {
          label: 'Go back',
          onClick: () => console.log('Go back'),
        },
      ],
      type: Toast.TYPE.NORMAL,
    });


  return <Button onClick={onClick}>Show toast</Button>;
}
```

#### Com link de informações adicionais

```js
function render() {
  const onClick = () =>
    Toast.showToast({
      title: 'TV Mode activated',
      description: 'Press ESC to exit',
      additionalInfoLink: {
        label: 'Learn more about TV Mode',
        to: 'https://discuss.newrelic.com/t/tv-mode-in-new-relic-one/92963',
      },
      type: Toast.TYPE.NORMAL,
    });


  return <Button onClick={onClick}>Show toast</Button>;
}
```

#### Notificação crítica de brinde

```js
function render() {
  const onClick = () =>
    Toast.showToast({
      title: 'Service unavailable',
      description: 'Your data could not be retrieved',
      actions: [
        {
          label: 'Retry',
          onClick: () => console.log('Retry'),
        },
      ],
      type: Toast.TYPE.CRITICAL,
    });


  return <Button onClick={onClick}>Update</Button>;
}
```
