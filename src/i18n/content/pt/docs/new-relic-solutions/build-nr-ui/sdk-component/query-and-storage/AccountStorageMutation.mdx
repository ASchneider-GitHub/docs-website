Salve, exclua ou atualize um documento com escopo de conta no NerdStorage. Para executar esta operação com êxito, o usuário deve ter as permissões corretas.

### Uso

```js
import { AccountStorageMutation } from 'nr1'
```

### Exemplos

#### Escrever documento

```js
AccountStorageMutation.mutate({
  accountId: 1,
  actionType: AccountStorageMutation.ACTION_TYPE.WRITE_DOCUMENT,
  collection: 'myCollection',
  documentId: 'myDocumentId',
  document: {
    name: 'John',
    surname: 'Doe',
  },
});
```

#### Excluir documento

```js
AccountStorageMutation.mutate({
  accountId: 1,
  actionType: AccountStorageMutation.ACTION_TYPE.DELETE_DOCUMENT,
  collection: 'myCollection',
  documentId: 'myDocumentId',
});
```

#### Excluir coleção

```js
AccountStorageMutation.mutate({
  accountId: 1,
  actionType: AccountStorageMutation.ACTION_TYPE.DELETE_COLLECTION,
  collection: 'myCollection',
});
```
