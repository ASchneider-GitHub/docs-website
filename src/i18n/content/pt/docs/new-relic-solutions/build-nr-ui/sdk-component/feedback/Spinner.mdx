Spinners são usados para indicar ao seu usuário que o conteúdo está carregando na tela.

### Uso

```js
import { Spinner } from 'nr1'
```

### Exemplos

#### Básico

```js
<Spinner />
```

#### Ponto

```js
<Spinner type={Spinner.TYPE.DOT} />
```

#### Em linha

```js
<Spinner inline />
```
