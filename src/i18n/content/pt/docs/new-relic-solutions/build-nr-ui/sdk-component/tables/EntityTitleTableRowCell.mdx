Renderiza uma célula da tabela de linhas mostrando o título de uma entidade específica. Esta célula toma o objeto entidade como filho, o que por sua vez é compatível com o objeto entidade NerdGraph (precisando, no mínimo, dos campos `name` e `reporting` ). Se puder ser alertado, você também deve consultar `alertSeverity`.

### Uso

```js
import { EntityTitleTableRowCell } from 'nr1'
```

### Exemplos

#### Básico

```js
function render() {
  const items = [
    {
      name: 'Login service',
      guid: 'MTIzNDU2fEZPT3xCQVJ8OTg3NjU0MzIx',
      alertSeverity: 'CRITICAL',
      reporting: true,
    },
  ];


  return (
    <Table items={items}>
      <TableHeader>
        <TableHeaderCell>Entity</TableHeaderCell>
      </TableHeader>


      {({ item }) => (
        <TableRow>
          <EntityTitleTableRowCell value={item} />
        </TableRow>
      )}
    </Table>
  );
}
```

#### Com valor adicional

```js
function render() {
  const items = [
    {
      name: 'Login service',
      guid: 'MTIzNDU2fEZPT3xCQVJ8OTg3NjU0MzIx',
      alertSeverity: 'CRITICAL',
      reporting: true,
    },
  ];


  return (
    <Table items={items} multivalue>
      <TableHeader>
        <TableHeaderCell>Entity</TableHeaderCell>
      </TableHeader>


      {({ item }) => (
        <TableRow>
          <EntityTitleTableRowCell value={item} additionalValue={item.guid} />
        </TableRow>
      )}
    </Table>
  );
}
```
