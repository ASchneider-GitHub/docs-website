Renderiza uma célula da tabela de linhas mostrando um usuário específico. Esta célula toma o objeto usuário como filho, o que por sua vez é compatível com o objeto usuário NerdGraph (precisando, no mínimo, dos campos `name` e `gravatar` ).

### Uso

```js
import { UserTableRowCell } from 'nr1'
```

### Exemplos

#### Exemplo 1

```js
function render() {
  const items = [
    {
      name: 'John Doe',
      gravatar:
        'https://secure.gravatar.com/avatar/79988df068cb4d952284a3395412dc4f',
    },
  ];


  return (
    <Table items={items}>
      <TableHeader>
        <TableHeaderCell>User</TableHeaderCell>
      </TableHeader>


      {({ item }) => (
        <TableRow>
          <UserTableRowCell value={item} />
        </TableRow>
      )}
    </Table>
  );
}
```
