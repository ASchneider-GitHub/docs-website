Quando um usuário escolhe uma ou mais opções fornecidas

O estado da caixa de seleção pode ser controlado pelas propriedades `checked` e `indeterminate` . Se nenhum deles for fornecido, a caixa de seleção gerenciará seu próprio estado.

Você pode ouvir as mudanças de estado da caixa de seleção por meio da propriedade `onChange` .

**Nota**: A configuração `checked` substituirá `defaultChecked` , pois coloca o componente em um estado controlado. `defaultChecked` só deve ser usado para caixas de seleção não controladas.

### Uso

```js
import { Checkbox } from 'nr1'
```

### Exemplos

#### Básico

```js
<div className="nr1-Docs-prettify">
  <Checkbox onChange={(event) => alert('Foo')} label="Foo" />
  <Checkbox indeterminate label="Bar" />
  <Checkbox checked disabled label="Baz" />
</div>
```

#### Com informações

```js
<Checkbox onChange={(event) => alert('Foo')} info="Info value" label="Foo" />
```

#### Com descrição

```js
<Checkbox
  onChange={(event) => alert('Foo')}
  description="Description value"
  label="Foo"
/>
```

#### Com mensagem inválida

```js
<Checkbox
  onChange={(event) => alert('Foo')}
  invalid="Invalid message value"
  label="Foo"
/>
```

#### Componente controlado

```js
class MyNerdlet extends React.PureComponent {
  constructor(props) {
    super(props);


    this.state = {
      isChecked: false,
    };


    this.onChange = this.onChange.bind(this);
  }


  onChange(event) {
    console.log(event);
    const isChecked = event.target.checked;


    this.setState({ isChecked });
  }


  render() {
    return (
      <Checkbox
        checked={this.state.isChecked}
        onChange={this.onChange}
        label="Foo"
      />
    );
  }
}
```
