`<Popover>` são sobreposições alternáveis para exibir pequenas sobreposições sob demanda.

O componente `<Popover>` exige que você forneça um acionador com `<PopoverTrigger>` que manipulará a abertura/fechamento da sobreposição, bem como o conteúdo da sobreposição por meio de `<PopoverBody />`.

### Uso

```js
import { Popover } from 'nr1'
```

### Exemplos

#### Básico

```js
<Popover>
  <PopoverTrigger>Trigger</PopoverTrigger>
  <PopoverBody>
    <BlockText>Body</BlockText>
  </PopoverBody>
</Popover>
```

#### Botão como gatilho

```js
<Popover>
  <PopoverTrigger>
    <Button>Trigger</Button>
  </PopoverTrigger>
  <PopoverBody>
    <BlockText>Body</BlockText>
  </PopoverBody>
</Popover>
```

#### TextField como gatilho

```js
<Popover>
  <PopoverTrigger>
    <TextField />
  </PopoverTrigger>
  <PopoverBody>
    <BlockText>Body</BlockText>
  </PopoverBody>
</Popover>
```

#### Com PopoverFooter

```js
<Popover>
  <PopoverTrigger>
    <Button>Trigger</Button>
  </PopoverTrigger>
  <PopoverBody>
    <Card style={{ width: '250px' }}>
      <CardBody>
        <HeadingText>Heading</HeadingText>
        <BlockText
          spacingType={[
            BlockText.SPACING_TYPE.MEDIUM,
            BlockText.SPACING_TYPE.NONE,
          ]}
        >
          Item content
        </BlockText>
        <BlockText>Item content</BlockText>
      </CardBody>
    </Card>
    <PopoverFooter style={{ textAlign: 'right' }}>Info message</PopoverFooter>
  </PopoverBody>
</Popover>
```

#### Popover controlado

```js
class ControlledPopover extends React.Component {
  constructor() {
    super(...arguments);


    this.state = {
      opened: false,
    };


    this._onChange = this._onChange.bind(this);
  }


  _onChange(evt, opened) {
    this.setState({ opened }, () =>
      console.log('controlled state', evt, this.state),
    );
  }


  render() {
    const { opened } = this.state;


    return (
      <Popover opened={opened} onChange={this._onChange}>
        <PopoverTrigger>
          <Button>Trigger</Button>
        </PopoverTrigger>
        <PopoverBody>
          <Card style={{ width: '250px' }}>
            <CardBody>
              <HeadingText>Heading</HeadingText>
              <BlockText
                spacingType={[
                  BlockText.SPACING_TYPE.MEDIUM,
                  BlockText.SPACING_TYPE.NONE,
                ]}
              >
                Item content
              </BlockText>
              <BlockText>Item content</BlockText>
            </CardBody>
          </Card>
          <PopoverFooter style={{ textAlign: 'right' }}>
            Info message
          </PopoverFooter>
        </PopoverBody>
      </Popover>
    );
  }
}
```
