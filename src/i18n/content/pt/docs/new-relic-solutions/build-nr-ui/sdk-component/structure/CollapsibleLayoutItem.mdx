Elemento filho do componente `<Layout>` usado apenas para áreas do lado esquerdo ou direito, com a capacidade de recolhê-las. Pode ser usado como um componente "controlado" (ou seja, passando a propriedade booleana `collapsed` ) ou, por padrão, ele gerenciará seu próprio estado (não) recolhido. É necessário um adereço `triggerType` , permitindo que você escolha os gatilhos embutidos – a área lateral incluirá um botão que permite ao usuário recolher a área e expandi-la novamente – ou personalizado, o que significa que você está usando seu próprio elemento de interface em outro lugar para controlar o estado (des)colapsado.

Consulte [a documentação `<Layout>` ](/docs/new-relic-solutions/build-nr-ui/sdk-component/structure/Layout/#examples)para ver exemplos de uso.

O componente `<Layout>` suporta apenas um filho de cada `type`, compartilhado entre `<CollapsibleLayoutItem>` e `<LayoutItem>`.

### Uso

```js
import { CollapsibleLayoutItem } from 'nr1'
```
