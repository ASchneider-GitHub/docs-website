Elemento filho do componente `<Layout>` .

Por padrão usa o tipo principal, para abranger toda a largura da área Layout. Dividir à esquerda e dividir à direita fornecem uma área fixa esquerda/direita. O layout suporta apenas um tipo de cada.

### Uso

```js
import { LayoutItem } from 'nr1'
```
