Fornece espaçamento para qualquer componente encapsulado. O espaçamento é fornecido como uma tupla que pode conter zero, um, dois, três ou quatro valores. Cada um desses valores vem da propriedade estática TYPE.

A estrutura da tupla é muito semelhante à forma como as propriedades `margin` ou `padding` funcionam: especificar um valor os replica nos quatro lados, dois em `top`/`bottom` e `left`/`right`, etc. Se você deseja omitir um valor específico, ao passar outros, você pode usar `OMIT` como espaço reservado. O espaço na lateral (ou laterais) onde é utilizado ficará intacto.

Outros componentes do SDK possuem recursos de espaçamento integrados. Se desejar replicar o mesmo recurso em seu próprio componente, você poderá usar `Spacing.add` e transmitir o componente; o componente retornado pelo método conterá a propriedade `type` bem como a estática.

### Uso

```js
import { Spacing } from 'nr1'
```

        `type` <h5>enum\[]</h5>
      </td>

      <td>
        Propriedade de espaçamento. O espaçamento é definido como uma tupla de zero a quatro valores, que segue as mesmas convenções das propriedades CSS como `margin` ou `padding`. Para omitir um valor, use `TYPE.OMIT`.

        <OptionReference array>
          Espaçamento.TYPE.EXTRA_LARGE, Espaçamento.TYPE.LARGE, Espaçamento.TYPE.MEDIUM, Espaçamento.TYPE.NONE, Espaçamento.TYPE.OMIT, Espaçamento.TYPE.SMALL,
        </OptionReference>

        Básico

        ```js
        <Spacing type={[Spacing.TYPE.MEDIUM]}>
          <Button>Spaced button</Button>
        </Spacing>;            
        ```

        Omitir margens

        ```js
        <Spacing type={[Spacing.TYPE.OMIT, Spacing.TYPE.LARGE]}>
          <Button>Spaced button (only left and right)</Button>
        </Spacing>; 
        ```
      </td>
