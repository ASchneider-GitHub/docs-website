Elemento filho do componente `<CardBody>` .

### Uso

```js
import { CardSection } from 'nr1'
```

### Props

      <td>
        `className` <h5>corda</h5>

      </td>
    </tr>

    <tr>
      <td>
        `collapsed` <h5>boleano</h5>
      </td>

      <td>
        Defina esta propriedade para controlar o estado recolhido do componente. Se definido, ele transforma o componente em um [componente controlado](https://facebook.github.io/react/docs/forms.html).
      </td>
    </tr>

    <tr>
      <td>
        `collapsible` <h5>boleano</h5>
      </td>

      <td>
        Defina esta propriedade para tornar o componente recolhível.
      </td>
    </tr>

    <tr>
      <td>
        `defaultCollapsed` <h5>boleano</h5>
      </td>

      <td>
        Se `true`, o estado inicial recolhido do cartão é `collapsed`.Útil quando você não deseja usar um [componente controlado](https://facebook.github.io/react/docs/forms.html).
      </td>
    </tr>

    <tr>
      <td>

      </td>

      <td>
        Esta função será chamada sempre que o usuário clicar no cabeçalho para recolher ou expandir o componente. Se você estiver controlando o estado do componente, use isso para definir o valor da propriedade `collapsed` .

        <FunctionDefinition

      </td>
    </tr>

        `testId` <h5>corda</h5>
      </td>

      <td>
        Adiciona um atributo `data-test-id` . Use-o para destino do componente em testes unitários e E2E. Para que um ID de teste seja válido, prefixe-o com seu ID nerdpack, seguido por um ponto. Por exemplo, `my-nerdpack.some-element`.

        **Nota:** Você pode não ver `data-test-id` atributo pois eles são removidos do DOM, para depurá-los passe um parâmetro de consulta `e2e-test` para a URL.
