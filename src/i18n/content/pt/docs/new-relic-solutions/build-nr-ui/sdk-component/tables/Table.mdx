Renderiza uma tabela com cabeçalho e linhas fixas. A tabela implementa o seguinte recurso:

* Layout flexível: os cabeçalhos das tabelas aceitam uma variedade de tamanhos para permitir layouts fluidos e fixos. Você pode encontrar mais informações sobre como personalizar a verificação de colunas `TableHeaderCell`.
* Classificação: os itens passados podem ser classificados internamente pela tabela de acordo com o estado atual de classificação da tabela. Você pode encontrar mais informações sobre classificação verificando `TableHeaderCell`.
* Seleção de linha: as linhas podem ser selecionadas através das caixas de seleção no lado direito. Você pode encontrar mais informações na proposta `selected` .
* Ações de linha: ações contextuais podem ser acionadas por linha, permitindo ao usuário obter funcionalidade sobre elas. Você pode encontrar mais informações sobre como adicioná-los em `TableRow`.
* Células personalizadas pré-definidas: alguns padrões muito comuns para células (título da entidade, métrica, etc.) já são fornecidos pela plataforma, então você só precisa retorná-los como parte da sua linha.
* Virtualização: as células só são renderizadas se forem mostradas na tela. Isso permite que a tabela funcione com um grande conjunto de dados quase sem penalidade de desempenho.

### Uso

```js
import { Table } from 'nr1'
```

#### Classificação de coluna única

```js
class Example extends React.Component {
  constructor(...args) {
    super(...args);


    this.state = {
      column: 0,
      sortingType: TableHeaderCell.SORTING_TYPE.NONE,
    };


    this.exampleItems = [
      { x: 'A', y: 0 },
      { x: 'B', y: 0 },
      { x: 'A', y: 1 },
    ];
  }


  _onClickTableHeaderCell(column, evt, { nextSortingType }) {
    if (column === this.state.column) {
      this.setState({ sortingType: nextSortingType });
    } else {
      this.setState({ column: column, sortingType: nextSortingType });
    }
  }


  render() {
    const sortingType0 =
      this.state.column === 0
        ? this.state.sortingType
        : TableHeaderCell.SORTING_TYPE.NONE;


    const sortingType1 =
      this.state.column === 1
        ? this.state.sortingType
        : TableHeaderCell.SORTING_TYPE.NONE;


    return (
      <Table items={this.exampleItems}>
        <TableHeader>
          <TableHeaderCell
            sortable
            sortingType={sortingType0}
            onClick={this._onClickTableHeaderCell.bind(this, 0)}
            value={({ item }) => item.x}
          >
            Column 1
          </TableHeaderCell>


          <TableHeaderCell
            sortable
            sortingType={sortingType1}
            onClick={this._onClickTableHeaderCell.bind(this, 1)}
            value={({ item }) => item.y}
          >
            Column 2
          </TableHeaderCell>
        </TableHeader>


        {({ item }) => (
          <TableRow>
            <TableRowCell>{item.x}</TableRowCell>
            <TableRowCell>{item.y}</TableRowCell>
          </TableRow>
        )}
      </Table>
    );
  }
}
```
