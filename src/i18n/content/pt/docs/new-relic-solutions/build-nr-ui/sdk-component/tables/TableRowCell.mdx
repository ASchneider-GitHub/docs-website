Envolve o conteúdo de uma célula da tabela localizada no corpo da tabela. Espera-se que este componente seja usado apenas como filho de `<TableRow>`. As células se encarregam automaticamente de elipsificar o conteúdo que não cabe e centralizá-lo verticalmente.

Este componente é a célula base. Para casos comuns, o SDK também fornece um conjunto de células adicionais para que elas sempre sejam pintadas de forma consistente na interface do usuário (por exemplo, nome da entidade com um status ou um usuário).

### Uso

```js
import { TableRowCell } from 'nr1'
```

### Exemplos

#### Célula de linha com links externos e informações adicionais

```js
function render() {
  const items = [
    {
      dashboardName: 'Staging service - Errors',
      createdBy: 'john@doe.com',
      url: '//fake-url.com/dashboard1',
    },
    {
      dashboardName: 'Package Registry - Performance',
      createdBy: 'jane@doe.com',
      url: '//fake-url.com/dashboard2',
    },
  ];
  return (
    <Table items={items} multivalue>
      <TableHeader>
        <TableHeaderCell>Name</TableHeaderCell>
        <TableHeaderCell>Url</TableHeaderCell>
      </TableHeader>
      {({ item }) => (
        <TableRow>
          <TableRowCell additionalValue={`Created by ${item.createdBy}`}>
            {item.dashboardName}
          </TableRowCell>
          <TableRowCell>
            <Link to={item.url}>Go to the dashboard</Link>
          </TableRowCell>
        </TableRow>
      )}
    </Table>
  );
}
```
