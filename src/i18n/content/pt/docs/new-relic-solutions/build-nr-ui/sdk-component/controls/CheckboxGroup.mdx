Um grupo de `<Checkbox>` botões. Os botões `<Checkbox>` podem ser filhos diretos ou descendentes do grupo de caixas de seleção.

**Nota**: A configuração `value` substituirá `defaultValue` pois coloca o grupo de caixas de seleção em um estado controlado. `value` não substituirá as caixas de seleção que tenham `checked` definido, pois `CheckboxGroup` assumirá o controle dos componentes `Checkbox` não controlados e não poderá substituir os controlados. No entanto, `value` substituirá `defaultChecked` em `Checkbox`.

O manipulador de eventos `onChange` para `CheckboxGroup` será acionado após qualquer manipulador de eventos `onChange` definido em componentes `Checkbox` individuais. No entanto, é altamente recomendável definir apenas um manipulador de eventos para todo o grupo em um `CheckboxGroup` controlado.

`Checkbox` os componentes **devem** ter um `value` definido para serem controlados por `CheckboxGroup`.

### Uso

```js
import { CheckboxGroup } from 'nr1'
```

### Exemplos

#### Básico

```js
<CheckboxGroup label="My checkbox group">
  <Checkbox label="Checkbox 1" />
  <Checkbox label="Checkbox 2" />
  <Checkbox label="Checkbox 3" />
</CheckboxGroup>
```

#### Com etiqueta e informações

```js
<CheckboxGroup label="My checkbox group" info="Info value">
  <Checkbox label="Checkbox 1" />
  <Checkbox label="Checkbox 2" />
  <Checkbox label="Checkbox 3" />
</CheckboxGroup>
```

#### Com etiqueta embutida

```js
<CheckboxGroup label="My checkbox group" labelInline>
  <Checkbox label="Checkbox 1" />
  <Checkbox label="Checkbox 2" />
  <Checkbox label="Checkbox 3" />
</CheckboxGroup>
```

#### Com descrição

```js
<CheckboxGroup label="My checkbox group" description="Description value">
  <Checkbox label="Checkbox 1" />
  <Checkbox label="Checkbox 2" />
  <Checkbox label="Checkbox 3" />
</CheckboxGroup>
```

#### Com mensagem inválida

```js
<CheckboxGroup label="My checkbox group" invalid="Invalid message value">
  <Checkbox label="Checkbox 1" />
  <Checkbox label="Checkbox 2" />
  <Checkbox label="Checkbox 3" />
</CheckboxGroup>
```

#### Com valores padrão

```js
<CheckboxGroup label="My checkbox group" defaultValue={['1', '2']}>
  <Checkbox label="Checkbox 1" value="1" />
  <Checkbox label="Checkbox 2" value="2" />
  <Checkbox label="Checkbox 3" value="3" />
</CheckboxGroup>
```

#### Componente controlado

```js
class MyNerdlet extends React.PureComponent {
  constructor(props) {
    super(props);


    this.state = {
      values: ['2'],
    };


    this.onChange = this.onChange.bind(this);
  }


  onChange(event, values) {
    this.setState({ values });
  }


  render() {
    return (
      <CheckboxGroup value={this.state.values} onChange={this.onChange}>
        <Checkbox label="Checkbox 1" value="1" />
        <Checkbox label="Checkbox 2" value="2" />
        <Checkbox label="Checkbox 3" value="3" />
      </CheckboxGroup>
    );
  }
}
```
