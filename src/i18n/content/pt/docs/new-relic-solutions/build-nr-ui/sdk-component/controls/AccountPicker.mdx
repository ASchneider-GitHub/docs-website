Recupera todas as contas e permite a seleção de contas por meio de dropdown e pesquisa. Para capturar a conta atualmente selecionada, você pode usar o ouvinte `onChange` .

Observe que a plataforma já contém seu próprio seletor de contas, que está localizado no cabeçalho da mesma. Este pode ser configurado para ser mostrado usando a API `nerdlet.setConfig` (passando `{ accountPicker: true }`), e seu valor pode ser lido usando `<PlatformStateContext.Consumer>`. Para referência completa, verifique estas API.

### Uso

```js
import { AccountPicker } from 'nr1'
```

### Exemplos

#### Básico

```js
class Example extends React.Component {
  constructor() {
    super(...arguments);


    this.state = { accountId: null };


    this.onChangeAccount = this.onChangeAccount.bind(this);
  }


  onChangeAccount(_, value) {
    alert(`Selected account: ${value}`);


    this.setState({ accountId: value });
  }


  render() {
    return (
      <AccountPicker
        value={this.state.accountId}
        onChange={this.onChangeAccount}
      />
    );
  }
}
```

#### Com etiqueta embutida

```js
class Example extends React.Component {
  constructor() {
    super(...arguments);


    this.state = { accountId: null };


    this.onChangeAccount = this.onChangeAccount.bind(this);
  }


  onChangeAccount(_, value) {
    alert(`Selected account: ${value}`);


    this.setState({ accountId: value });
  }


  render() {
    return (
      <AccountPicker
        label="Account"
        labelInline
        value={this.state.accountId}
        onChange={this.onChangeAccount}
      />
    );
  }
}
```
