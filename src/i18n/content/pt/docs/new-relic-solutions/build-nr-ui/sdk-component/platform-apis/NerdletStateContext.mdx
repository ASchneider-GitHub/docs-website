[Contexto de reação](https://reactjs.org/docs/context.html#when-to-use-context) para acessar o estado da URL do nerdlet.

### Uso

```js
import { NerdletStateContext } from 'nr1'
```

### Exemplos

#### Básico

```js
<NerdletStateContext.Consumer>
  {(nerdletState) => <pre>{JSON.stringify(nerdletState, null, 4)}</pre>}
</NerdletStateContext.Consumer>
```
