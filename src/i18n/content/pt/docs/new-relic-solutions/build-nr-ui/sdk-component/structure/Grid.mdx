Grid é um sistema de layout bidimensional, o que significa que ele pode lidar com colunas e linhas, o que ajuda a projetar a interface do usuário baseada em grade, ao contrário de `<Stack>` , que é basicamente um sistema unidimensional.

### Uso

```js
import { Grid } from 'nr1'
```
