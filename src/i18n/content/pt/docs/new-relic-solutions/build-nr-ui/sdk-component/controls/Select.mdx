O componente `<Select>` é uma variante do `<Dropdown>` , especialmente destino para ser usado em componentes de formulário. As principais diferenças são:

* Só pode funcionar em modo declarativo, ou seja, não pode ser virtualizado passando uma função filha.
* Cada `<SelectItem>` aceita uma propriedade `value` , que será comparada com o `value` passado para `<Select>`, e a opção correta será selecionada. Ao contrário de um elemento DOM `<SELECT>` tradicional, qualquer referência pode ser usada como valor, tanto primitivos quanto objetos.
* Os itens não aceitam um `onClick`. Em vez disso, você obterá o valor da opção selecionada recebendo-o de um método `onChange` .

O componente é sempre controlado, o que significa que você precisa atualizar seu `value` a partir de um retorno de chamada `onChange` para que ele reflita a propriedade recém-selecionada.

### Uso

```js
import { Select } from 'nr1'
```

### Exemplos

#### Básico

```js
<Select onChange={(evt, value) => alert(value)}>
  <SelectItem value="a">Value is "a"</SelectItem>
  <SelectItem value="b">Value is "b"</SelectItem>
  <SelectItem value="c">Value is "c"</SelectItem>
</Select>
```

#### Com etiqueta e informações

```js
<Select label="Items" info="Info value" onChange={(evt, value) => alert(value)}>
  <SelectItem value="a">Value is "a"</SelectItem>
  <SelectItem value="b">Value is "b"</SelectItem>
  <SelectItem value="c">Value is "c"</SelectItem>
</Select>
```

#### Com etiqueta embutida

```js
<Select label="Items" labelInline onChange={(evt, value) => alert(value)}>
  <SelectItem value="a">Value is "a"</SelectItem>
  <SelectItem value="b">Value is "b"</SelectItem>
  <SelectItem value="c">Value is "c"</SelectItem>
</Select>
```

#### Com descrição

```js
<Select description="Description value" onChange={(evt, value) => alert(value)}>
  <SelectItem value="a">Value is "a"</SelectItem>
  <SelectItem value="b">Value is "b"</SelectItem>
  <SelectItem value="c">Value is "c"</SelectItem>
</Select>
```

#### Com mensagem inválida

```js
<Select invalid="Invalid message value" onChange={(evt, value) => alert(value)}>
  <SelectItem value="a">Value is "a"</SelectItem>
  <SelectItem value="b">Value is "b"</SelectItem>
  <SelectItem value="c">Value is "c"</SelectItem>
</Select>
```

#### Componente controlado

```js
class ControlledSelect extends React.Component {
  constructor() {
    super(...arguments);


    this.state = {
      value: null,
    };


    this._onChange = this._onChange.bind(this);
  }


  _onChange(event, value) {
    this.setState({ value });
  }


  render() {
    return (
      <Select onChange={this._onChange} value={this.state.value}>
        <SelectItem value="1">Item 1</SelectItem>
        <SelectItem value="2">Item 2</SelectItem>
        <SelectItem value="3">Item 3</SelectItem>
      </Select>
    );
  }
}
```
