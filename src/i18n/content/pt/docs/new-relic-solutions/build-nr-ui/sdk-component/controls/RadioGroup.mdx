Um grupo de `<Radio>` botões. Os botões `<Radio>` podem ser filhos diretos ou descendentes do grupo de rádios. Os botões `<Radio>` dentro de um grupo de opções devem ter um `value` exclusivo atribuído.

Depois que um grupo de rádio for estabelecido, selecionar qualquer `<Radio>` nesse grupo desmarcará automaticamente qualquer `<Radio>` atualmente selecionado no grupo.

**Nota**: A configuração `value` substituirá `defaultValue` pois coloca o `RadioGroup` em um estado controlado. `value` não substituirá `Radio` componentes que tenham `checked` definido, pois `RadioGroup` assumirá o controle dos componentes `Radio` não controlados e não poderá substituir os controlados.

O manipulador de eventos `onChange` para `RadioGroup` será acionado após qualquer manipulador de eventos `onChange` definido em componentes `Radio` individuais. No entanto, é altamente recomendável definir apenas um manipulador de eventos para todo o grupo em um `RadioGroup` controlado.

### Uso

```js
import { RadioGroup } from 'nr1'
```

### Exemplos

#### Básico

```js
<RadioGroup defaultValue="2">
  <Radio label="Radio 1" value="1" />
  <Radio label="Radio 2" value="2" />
  <Radio label="Radio 3" value="3" />
</RadioGroup>
```

#### Com etiqueta e informações

```js
<RadioGroup label="Radio Group" info="Info value" defaultValue="2">
  <Radio label="Radio 1" value="1" />
  <Radio label="Radio 2" value="2" />
  <Radio label="Radio 3" value="3" />
</RadioGroup>
```

#### Com etiqueta embutida

```js
<RadioGroup label="Radio Group" labelInline defaultValue="2">
  <Radio label="Radio 1" value="1" />
  <Radio label="Radio 2" value="2" />
  <Radio label="Radio 3" value="3" />
</RadioGroup>
```

#### Com descrição

```js
<RadioGroup
  label="Radio Group"
  description="Description value"
  defaultValue="2"
>
  <Radio label="Radio 1" value="1" />
  <Radio label="Radio 2" value="2" />
  <Radio label="Radio 3" value="3" />
</RadioGroup>
```

#### Com mensagem inválida

```js
<RadioGroup
  label="Radio Group"
  invalid="Invalid message value"
  defaultValue="2"
>
  <Radio label="Radio 1" value="1" />
  <Radio label="Radio 2" value="2" />
  <Radio label="Radio 3" value="3" />
</RadioGroup>
```

#### Componente controlado

```js
class MyNerdlet extends React.PureComponent {
  constructor(props) {
    super(props);


    this.state = {
      selectedValue: '2',
    };


    this.onChange = this.onChange.bind(this);
  }


  onChange(event, value) {
    this.setState((state) => {
      return { selectedValue: value };
    });
  }


  render() {
    return (
      <RadioGroup value={this.state.selectedValue} onChange={this.onChange}>
        <Radio label="Radio 1" value="1" />
        <Radio label="Radio 2" value="2" />
        <Radio label="Radio 3" value="3" />
      </RadioGroup>
    );
  }
}
```
