<Callout variant="important" title="AVISO DE EOL">
  A API de Parceria atingiu o fim de sua vida útil (EOL) em março de 2023 e não tem mais suporte nem está disponível para uso.
</Callout>

Nossa API de parceria permitiu que parceiros New Relic e contas New Relic configuradas como [parcerias de clientes](/docs/new-relic-partnerships/partner-integration-guide/partner-account-maintenance/partnership-accounts-users-subscriptions/#partnerships) gerenciassem contas, usuários e configurações relacionadas à assinatura.

Como a API de Parceria não está mais disponível, você pode usar nossa API Nerdgraph para realizar muitas das mesmas tarefas:

* [Gerenciar contas](/docs/apis/nerdgraph/examples/manage-accounts-nerdgraph)
* [Gerenciar usuário](/docs/apis/nerdgraph/examples/nerdgraph-manage-users)
* [Gerenciar grupos de usuários](/docs/apis/nerdgraph/examples/nerdgraph-manage-groups)
