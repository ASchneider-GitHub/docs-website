Seja uma consulta ad-hoc ou uma consulta salva por meio [de arquivos NRQL](#nrql-files), a capacidade de executar consultas diretamente do seu IDE oferece acesso poderoso a todos os dados de desempenho que a New Relic possui sobre seus serviços. Se ainda não estiver familiarizado com NRQL, você pode aprender mais sobre [a linguagem de consulta New Relic](/docs/nrql/get-started/introduction-nrql-new-relics-query-language).

Clique no ícone “consulta seus dados” na navegação global do CodeStream para acessar o criador de consulta.

Com base nos dados que você especificou, o criador de consulta seleciona o tipo de gráfico que exibe seus resultados de forma mais eficaz. No entanto, você pode usar o dropdown acima dos resultados para escolher entre outros tipos de gráfico disponíveis para apresentar seus dados no formato que deseja usar. Você também pode exportar os dados no formato CSV ou JSON, o que abrirá uma nova guia do editor e exibirá os dados no formato selecionado.

## Arquivos NRQL [#nrql-files]

The dropdown above the query builder, at the right side, allows you to select from queries you've recently run. Another way to save queries, though, is to add a file with a `.nrql` extension to your repository. You can save your queries to this file, and even share those queries with your teammates by commiting and pushing that file. Just click the "Run" link in the CodeLens above each query to run the query.
