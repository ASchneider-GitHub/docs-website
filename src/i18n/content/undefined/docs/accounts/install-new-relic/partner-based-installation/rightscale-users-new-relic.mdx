[RightScale](https://www.rightscale.com/) é uma solução de software como serviço (SaaS) capaz de hospedar aplicativos da web. As linguagens suportadas para [New Relic](http://support.rightscale.com/27-Partners/New_Relic) e RightScale incluem Java, Ruby, .NET, PHP e Python.

Para usar o New Relic por meio da [parceria RightScale](http://www.rightscale.com/partners/isv/new-relic.php), você precisa de uma conta New Relic. Antes de começar, certifique-se de [ativar o acesso à API](/docs/apis/rest-api-v2/requirements/new-relic-rest-api-v2-getting-started).

## Adicionar New Relic RightScripts [#add-rpm]

Antes de poder visualizar os dados do New Relic no RightScale, adicione o New Relic RightScript apropriado ao seu ServerTemplates:

1. No RightCloud, selecione **Design > MultiCloud Marketplace > ServerTemplates**.
2. Importe o [New Relic Performance Monitoring Toolbox](http://www.rightscale.com/library/server_templates/New-Relic-Performance-Monitori/lineage/17048 "O link abre em uma nova janela") ServerTemplate para sua visualização **Local** .

Depois de importar o ServerTemplate, você pode usar os New Relic RightScripts associados à sua conta e adicioná-los aos ServerTemplates conforme apropriado para o idioma do seu agente.

<figcaption>
  Adicione o **APP New Relic RPM for PHP** RightScript como um **boot script** ao ServerTemplate que está sendo usado para executar seus servidores de aplicativos PHP. Isso implantará automaticamente o agente New Relic quando um novo servidor for lançado com esse ServerTemplate.
</figcaption>

Se o seu aplicativo PHP já estiver em execução em um servidor, você poderá adicioná-lo como um **script operacional** na guia **Scripts** e executar a ação no servidor em execução.

Certifique-se de definir o seguinte valor de entrada para o novo script:

```
NR_License_Key: Enter your New Relic license key.
```

Você pode inserir manualmente o <InlinePopover type="licenseKey"/>ou criar uma credencial para ele.

## Visualizar dados de monitoramento do APM [#view]

Ao visualizar os dados que o New Relic monitora, você verá apenas os nomes dos aplicativos que estão nos servidores onde você executou o New Relic RightScript. Um **host** é um servidor e uma **instância** é uma instância de um aplicativo.

<img
  title="screen nr rightscale applist.png"
  alt="screen nr rightscale applist.png"
  src={accountsRightscaleAppList}
/>

<figcaption>
  **Usuário RightScale:** Exemplo de aplicativos que você pode visualizar no New Relic.
</figcaption>

Para visualizar seus dados do New Relic:

1. Vá para **Reports > New Relic**.
2. Na lista de aplicativos em sua implantação do New Relic RightScript, selecione um nome de aplicativo.
3. No seu primeiro login, certifique-se de [alterar sua senha](/docs/subscriptions/changing-passwords-and-user-preferences): No [menu de usuário \[user menu\]](/docs/accounts/accounts-billing/general-account-settings/intro-account-settings) na UI do New Relic, selecione **User preferences**.
4. Use as opções de menu do New Relic para visualizar diferentes dados e gráficos de desempenho.

## Atualize sua assinatura do New Relic [#upgrade]

Para clientes RightScale, todas as novas contas começam com uma avaliação gratuita de 14 dias do [recurso de assinatura paga](http://newrelic.com/application-monitoring/pricing) da New Relic. Se você quiser atualizar sua conta gratuita para uma assinatura paga, envie um e-mail para `rightscale@newrelic.com`. Você não pode atualizar sua assinatura do New Relic no RightScale.
